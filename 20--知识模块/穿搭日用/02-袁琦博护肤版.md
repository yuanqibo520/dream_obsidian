## 前言

### 目标与期望

> 目标：安全提亮肤色、均匀肤色、减轻暗沉与浅表色斑，不追求急速漂白。
> 预期：4-12 周出现可见改善，显著变化通常需 3 个月以上。

### 核心原则

- 防晒为首要步骤（每天、全年执行）。
- 温和逐步上成分，任何刺激停用并复查。
- 记录对比照，固定光线与角度。
- 避免来路不明的速效“漂白”产品（含汞、强类固醇等）。

---
### 8 周落地时间表

- 周 1-2：建立基础（温和洁面、保湿、防晒）。早上开始使用维生素 C。
- 周 3-4：若无不良反应，晚间引入烟酰胺或传明酸精华（可交替晚间使用）。开始每周 1 次轻度 AHA 去角质。
- 周 5-6：如皮肤耐受，夜间引入低浓度视黄醇（每周 1-2 次），并继续 AHA（1 次 / 周，或视情况 2 次 / 周）。
- 周 7-8：将视黄醇频率逐步提升到每周 3 次（若耐受），继续维持维 C + 烟酰胺 / 传明酸，第 8 周做照片对比与效果评估。

---
### 每周示例日程

- 周一：早维 C / 晚传明酸
- 周二：早维 C / 晚保湿修复（神经酰胺、透明质酸）。
- 周三：早维 C / 晚视黄醇（低频）
- 周四：早维 C / 晚烟酰胺
- 周五：早维 C / 晚保湿修复（AHA 去角质夜）
- 周末：补水面膜、休息成分让皮肤修复

---
### 每日早 / 晚流程

##### 早晨

1. 温和洁面（早晨可用清水或温和洁面）。
2. 保湿化妆水 / 爽肤水（可选）。
3. 维生素 C 精华（早上，推荐 10-15% L-抗坏血酸或稳定衍生物）。
4. 轻薄保湿乳 / 面霜（含烟酰胺更佳）。

##### 晚上

1. 温和洁面
2. 目标精华（传明酸 / 熊果苷 / 烟酰胺等交替使用）。
3. 视黄醇类（低浓度，见时间表）仅夜间使用且与 AHA 同晚避免同用。
4. 保湿修复（神经酰胺、透明质酸）。
5. 每周去角质 1 次起（AHA 低浓度 5-10% 或酵素），根据耐受逐步调整频率。

##### 补充注意

- 先在耳后或下颌做 48-72 小时贴试（patch test）。
- 出现严重刺痛、大片红肿或持续色沉加重则停用并咨询皮肤科。

---
### 推荐安全成分与使用频率（大学生友好）

- [ ] 维生素 C：早上使用（10-15% 或稳定型衍生物），抵抗氧化并提亮。
- [ ] 烟酰胺（Niacinamide）：2-5%，有助于淡斑、控油与修复屏障。
- [ ] 传明酸（Tranexamic Acid）：约 2%，针对色素沉着有效。
- [ ] 熊果苷（Arbutin）：温和抑黑，1-2%。
- [ ] 壬二酸（Azelaic Acid）：5-10%，对痘印与色素有帮助，敏感皮可用。
- [ ] AHA（果酸，例：乳酸 / 乙醇酸）：家用低浓度 5-10%，1 次 / 周起，视耐受增加到 2 次 / 周。
- [ ] BHA（水杨酸）：0.5-2%，油痘肌用于控油与清理毛孔。
- [ ] 视黄醇（Retinol）：低浓度 0.25-0.5% 起步，每周 1-2 次夜间，耐受后逐步增加频率。
- [ ] 保湿：透明质酸、神经酰胺、甘油等，夜间注重修复。

---
### 必须避免与高风险物质

- 严禁使用含汞的“速白”膏或来路不明的激素类面霜（可导致严重副作用）。
- 不要使用高浓度自配氢醌或长期无医嘱使用氢醌（可能致 ochronosis）。
- 避免未经许可的注射类美白（如来路不明的谷胱甘肽静脉注射等）。

### 记录与评估

- 固定光线与角度每周拍照对比（每两周明显变化）。
- 若有持续顽固色斑，先到皮肤科做专业诊断（排除炎症后色素沉着、真皮沉着等）。
