## 1. 基础原则

- 大多数简易调饮可以拆成**基酒 + 稀释液/气泡 + 酸 + 甜 + 冰**。
- **冰块不是装饰**：冰块负责降温、稀释和让酒精感变柔。高球杯建议装到 7-8 分满。
- **气泡最后加**：汤力水、苏打水、强爽、果味气泡饮最后倒，轻轻提拉搅拌，避免跑气。
- **低价酒优先做混饮**：
	价格低、风味简单的伏特加/金酒适合加橙汁、汤力水、茶饮；
	高价威士忌先净饮或加冰，确认风味后再决定是否调饮。

**推荐学习路径**：
1. **入门长饮**：螺丝起子、金汤力、自由古巴、威士忌嗨棒。
2. **需要控制酸甜**：龙舌兰日出、莫吉托、玛格丽特、莫斯科骡子。
3. **需要摇和过滤**：大都会、边车、盘尼西林。
4. **烈酒搅拌型**：尼格罗尼、干马天尼、曼哈顿。
5. **特殊技巧**：B-52 练分层，长岛冰茶练多基酒比例控制。

## 4. 推荐配方表
**高风险提示**：
- **长岛冰茶**：多基酒组合，入口像甜茶但酒精强，不建议连续饮用。
- **干马天尼**：几乎是纯烈酒型，适合小口慢饮。
- **曼哈顿**：烈酒搅拌型，酒感明显，苦精不要过量。
- **B-52**：家庭练习不建议点火，只练分层即可。

### 4.1 螺丝起子 Screwdriver

| 项目   | 内容                         |
| ---- | -------------------------- |
| 风味   | 橙香、甜酸、酒感低                  |
| 难度   | 入门                         |
| 杯型   | 高球杯                        |
| 配料   | 伏特加 45 ml、橙汁 120-150 ml、冰块 |
| 工具   | 量酒器、高球杯、吧勺                 |
| 核心技巧 | 直调、控甜酸                     |
| 可选   | 橙片 1 片、柠檬汁 5 ml            |
| 做法   | 杯中加冰，倒入伏特加，再倒橙汁，轻搅 5-8 圈   |
| 成本估算 | 约 6-12 元/杯                 |

**制作教程**：
1. 高球杯加冰到 7-8 分满。
2. 倒入伏特加 45 ml。
3. 加入橙汁 120-150 ml。
4. 用吧勺轻搅 5-8 圈。
5. 可放橙片装饰。

**严谨建议**：新手建议从 45 ml 伏特加开始；如果橙汁偏甜，加 5 ml 柠檬汁会更清爽。

### 4.2 金汤力 Gin & Tonic

| 项目 | 内容 |
|---|---|
| 风味 | 清爽、微苦、香料感 |
| 难度 | 入门 |
| 杯型 | 高球杯 |
| 配料 | 金酒 45 ml、汤力水 120-150 ml、冰块 |
| 工具 | 量酒器、高球杯、吧勺 |
| 核心技巧 | 控气泡、轻搅 |
| 可选 | 青柠角 1 块、柠檬皮 |
| 做法 | 杯中加满冰，倒入金酒，沿杯壁加入汤力水，轻轻提拉搅拌 2-3 次 |
| 成本估算 | 约 8-16 元/杯 |

**制作教程**：
1. 高球杯加满冰。
2. 倒入金酒 45 ml。
3. 沿杯壁慢慢加入汤力水 120-150 ml。
4. 用吧勺轻轻提拉 2-3 次。
5. 挤入青柠角或放柠檬皮装饰。

**严谨建议**：不要剧烈搅拌，汤力水跑气后会变甜腻。金酒香气越明显，柠檬皮越适合作为装饰。

### 4.3 自由古巴 Cuba Libre

| 项目 | 内容 |
|---|---|
| 风味 | 可乐甜香、青柠清爽、朗姆酒感轻 |
| 难度 | 入门 |
| 杯型 | 高球杯 |
| 配料 | 白朗姆 50 ml、可乐适量、青柠汁 10-15 ml、冰块 |
| 工具 | 量酒器、高球杯、吧勺 |
| 核心技巧 | 直调、控气泡 |
| 可选 | 青柠角 1-2 块 |
| 做法 | 高球杯加满冰，倒入白朗姆和青柠汁，补可乐至 8-9 分满，轻轻提拉搅拌 |
| 成本估算 | 约 7-14 元/杯 |

**制作教程**：
1. 高球杯加冰到 7-8 分满。
2. 倒入白朗姆 50 ml。
3. 加入青柠汁 10-15 ml。
4. 缓慢补入可乐。
5. 用吧勺轻轻提拉 2-3 次。
6. 放入青柠角装饰。

**严谨建议**：青柠汁是自由古巴的关键。可乐已经很甜，青柠汁建议从 10 ml 起步。

### 4.4 威士忌嗨棒 Whisky Highball

| 项目 | 内容 |
|---|---|
| 风味 | 清爽、麦芽或波本香、气泡感明显 |
| 难度 | 入门 |
| 杯型 | 高球杯 |
| 配料 | 威士忌 45 ml、苏打水 100-150 ml、冰块 |
| 工具 | 量酒器、高球杯、吧勺 |
| 核心技巧 | 冰杯预冷、控气泡 |
| 可选 | 柠檬皮 1 条 |
| 做法 | 高球杯加满冰，倒入威士忌，搅冷后沿杯壁加入苏打水，轻轻提拉 1-2 次 |
| 成本估算 | 约 8-18 元/杯 |

**制作教程**：
1. 高球杯加满冰，先搅几圈让杯子降温。
2. 倒掉杯中融出的多余水。
3. 倒入威士忌 45 ml。
4. 再搅 5-8 圈，让威士忌充分降温。
5. 沿杯壁加入苏打水 100-150 ml。
6. 用吧勺轻轻提拉 1-2 次。
7. 可挤一条柠檬皮油后放入杯中。

**严谨建议**：嗨棒看似简单，关键是冰足、杯冷、苏打水气足。不要剧烈搅拌，否则气泡会快速消失。

### 4.5 龙舌兰日出 Tequila Sunrise

| 项目 | 内容 |
|---|---|
| 风味 | 橙香明显、酸甜平衡、红石榴糖浆带出渐层 |
| 难度 | 入门 |
| 杯型 | 高球杯 |
| 配料 | 龙舌兰 40 ml、橙汁 120 ml、柠檬汁 10 ml、红石榴糖浆 10 ml、冰块 |
| 工具 | 量酒器、高球杯、吧勺 |
| 核心技巧 | 红石榴糖浆分层 |
| 可选 | 橙片 1 片、樱桃 1 颗 |
| 做法 | 高球杯加冰，倒入龙舌兰、橙汁和柠檬汁，轻搅；最后沿杯壁慢慢倒入红石榴糖浆形成渐层 |
| 成本估算 | 约 10-18 元/杯 |

**制作教程**：
1. 高球杯加冰到 7-8 分满。
2. 倒入龙舌兰 40 ml。
3. 加入橙汁 120 ml 和柠檬汁 10 ml，轻搅 5-8 圈。
4. 红石榴糖浆 10 ml 贴杯壁慢慢倒入，不要再搅拌。
5. 静置 10-20 秒，杯底变红，上层保持橙色。

**严谨建议**：红石榴糖浆是视觉关键，不建议一开始就搅散。柠檬汁不要超过 15 ml，否则会压住橙香。

### 4.6 莫吉托 Mojito

| 项目 | 内容 |
|---|---|
| 风味 | 青柠清酸、薄荷清凉、朗姆酒感轻快 |
| 难度 | 进阶 |
| 杯型 | 高球杯 |
| 配料 | 白朗姆 45 ml、青柠汁 20 ml、糖浆 10-15 ml、薄荷叶 8-12 片、苏打水 90-120 ml、冰块 |
| 工具 | 量酒器、高球杯、捣棒、吧勺 |
| 核心技巧 | 薄荷轻压、控气泡 |
| 可选 | 青柠角 2-3 块、薄荷枝 1 枝 |
| 做法 | 杯中放薄荷叶、青柠汁和糖浆，轻压出香气；加冰，倒入白朗姆，补苏打水，轻轻提拉搅拌 |
| 成本估算 | 约 9-16 元/杯 |

**制作教程**：
1. 高球杯中放薄荷叶、青柠汁和糖浆。
2. 用捣棒轻压薄荷 3-5 下，只压出香气。
3. 加冰到 7-8 分满。
4. 倒入白朗姆 45 ml。
5. 补入苏打水 90-120 ml。
6. 用吧勺从杯底轻轻提拉 2-3 次。

**严谨建议**：薄荷不要捣碎，捣太碎会带出草腥和苦味。苏打水最后加，搅拌要轻。

### 4.7 玛格丽特 Margarita

| 项目 | 内容 |
|---|---|
| 风味 | 酸爽、橙香、龙舌兰草本感明显 |
| 难度 | 进阶 |
| 杯型 | 玛格丽特杯或古典杯 |
| 配料 | 龙舌兰 40 ml、君度 20 ml、青柠汁 20 ml、盐适量、冰块 |
| 工具 | 量酒器、雪克杯、隔冰器、压汁器 |
| 核心技巧 | 盐边、摇和过滤 |
| 可选 | 青柠片 1 片 |
| 做法 | 杯口抹青柠后蘸盐；雪克杯加冰，倒入龙舌兰、君度和青柠汁，摇冷后过滤入杯 |
| 成本估算 | 约 12-22 元/杯 |

**制作教程**：
1. 用青柠片擦杯口外沿，轻蘸盐做半圈盐边。
2. 雪克杯中加冰。
3. 倒入龙舌兰 40 ml、君度 20 ml、青柠汁 20 ml。
4. 摇和 10-15 秒。
5. 过滤倒入杯中。

**严谨建议**：盐边不要太厚。青柠汁必须新鲜，瓶装汁会让玛格丽特显得尖酸。

## 6. 操作流程

1. 提前 4 小时冻冰，或直接买袋装冰。
2. 橙汁、茶、椰子水、汤力水提前冷藏。
3. 先按配方做一杯试饮，确认甜度和酒感。
4. 聚会时只开放 2-3 个固定菜单，避免材料乱混。
5. 每杯记录基酒用量，不凭手感倒酒。
6. 准备足量饮用水和无酒精饮料。

## 7. 风味调整公式

| 问题 | 调整方式 |
|---|---|
| 太甜 | 加柠檬汁 5 ml，或补无糖苏打/茶 |
| 太烈 | 加冰、加软饮 30-60 ml |
| 太淡 | 下次增加基酒 5-10 ml，不建议当杯反复加酒 |
| 太苦 | 减少汤力水比例，换更甜的汤力水，或加橙汁 15 ml |
| 没香气 | 加柠檬皮/青柠角，或换更香的金酒 |
| 跑气 | 气泡饮最后加，减少搅拌 |
