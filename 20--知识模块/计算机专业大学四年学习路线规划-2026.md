## 大一：基础筑基+前沿启蒙
---
### 目标
- 打牢计算机与编程基础；
- 形成良好的学习与实践习惯；
- 了解计算机各领域方向。
#### 核心课程
- 数学类：
	- [ ] 高等数学[[高数上下]]
	- [ ] 离散数学[[离散数学]]
	- [ ] 线性代数（重点练矩阵运算）[[线性代数]]
	- [ ] 概率论与数理统计（AI算法核心基础）[[统计学]]
- 编程类：
	- [ ] Python（优先掌握，AI/数据分析标配）
	- [ ] C语言（重点练语法、数据结构落地（如链表、哈希表））[[20--知识模块/C语言/基本语言代码|基本语言代码]]
	- [ ] 数据结构与算法（重点：树、图、动态规划）
	- [ ] ==计算机组成原理==
	- [ ] Linux操作系统[[Linux开发]]
	- [ ] 通识课：英语、物理
---

#### 实践动作
- 工具掌握：
	- [ ] VSCode[[奇技淫巧]]
	- [ ] Cursor/Claude
	- [x] Git版本控制[[Git 基本代码使用手册]]
	- [x] coze部署智能体
	- [ ] 办公软件

#### 关键提醒
- 数学是AI/算法岗的“硬门槛”，避免只学编程忽略数学；
- 利用AI工具（如Copilot）辅助代码调试，但需理解底层逻辑，拒绝“AI代笔”。
- 每周学习时间40-50小时，保持GPA在3.5以上。目标：建立自信，养成自学习惯。
- LeetCode入门题，目标每周10题。


## 大二：核心技术+方向试水
---
#### 核心目标
- 掌握计算机核心原理（组成、OS、网络），能独立调试系统级问题。
- 深度试水1个方向（AI或云原生），积累“技术+场景”经验。
- 构建初步作品集（Portfolio），至少2-3个可展示项目。
- 考取2-3个证书，提升简历辨识度。
- 养成AI协同编程习惯（用Cursor/Claude等工具加速开发，但核心逻辑自己设计）。
---
#### 核心课程
- **必修核心**：
	- [ ] 计算机组成原理（理解CPU/内存/IO）。
	- [ ] 操作系统（进程/内存管理，Linux实践）。
	- [ ] 计算机网络（TCP/IP、HTTP/HTTPS）。
	- [ ] 数据库系统（SQL/MySQL + NoSQL/MongoDB入门）。
	- [ ] 软件工程（需求分析、设计模式）。
- **方向选修**（选1深耕，其他浅尝）：
	- [ ] AI方向：机器学习基础（Scikit-learn/PyTorch），NLP入门（Transformer）。
	- [ ] 云原生方向：Docker/K8s入门，Go语言（高并发后端）。
- **编程进阶**：Python熟练 + Go/Java二选一（AI选Python+PyTorch，云选Go）。

| 方向         | 为什么推荐                      | 学习门槛           | 未来机会         |
| ---------- | -------------------------- | -------------- | ------------ |
| AI应用工程（首选） | 2027年AI岗位爆炸增长，用大模型降维打击传统开发 | 中等（Python基础即可） | 高薪+创业潜力★★★★★ |
| 云原生+Go（稳）  | 稳定就业，云厂商/大厂后端需求大           | 中等（需学新语言）      | 好找工作★★★★     |

---
#### 实践动作
- 技术落地：
	- 用PyTorch/Go实现简单项目（如AI：图像分类模型；云：Docker部署Web服务）
	- 算法优化：Big O分析+准备面试题（leetCode）。
	- 云平台实践：AWS或阿里云免费账户，部署1个应用。
- **项目积累**（至少完成2个，GitHub开源）：
	- 小项目：RAG知识库聊天机器人（AI）或Go微服务（云）。
	- 中型项目：完整Web应用（React + Node.js + LLM集成）或简单ML模型（TensorFlow图像分类）。
	- 项目文档及部署指南
	- 竞赛：参加Hackathon或Kaggle入门比赛。
	- 暑假实习：软件开发实习，或AI初创公司助理（目标：积累简历经验）。
- 证书入门：考取阿里云ACA、华为云HCIA等基础认证，提升简历辨识度。
--- 
#### 证书考取安排
每个证书准备1-2个月，结合课程学习时间准备
- **大二上（10-12月）**：
  - **阿里云ACA（Associate Cloud Architect）**：云原生基础认证。准备：结合Docker/K8s学习，官网免费教程+模拟题。考取时间：12月。益处：云方向简历加分。
  - **Coursera Google IT Support**：IT基础证书。准备：边学OS/网络，1个月课程。考取时间：11月。益处：通用技能，免费审计。
- **大二下（3-5月）**：
  - **IBM Data Science Professional Certificate**（Coursera）：AI/ML入门。准备：结合机器学习课程，2个月项目实践。考取时间：5月。益处：AI方向敲门砖，简历亮点。
  - **可选：华为云HCIA（AI方向）**：如果选AI，准备1个月。考取时间：4月。益处：国产厂商认可高。
- **总体Tips**：总花费<500元（Coursera订阅）。优先在线考，边学边考。证书上传LinkedIn/GitHub，提升曝光。
---
#### 时间线安排（完整一年）
- **大二上（9月-1月）**：
  - 9-10月：核心课程起步（组成/OS/网络），主方向入门（AI: PyTorch；云: Go/Docker）。启动小项目。
  - 11月：考Google IT Support。算法刷题。
  - 12月：考阿里云ACA。完成1个项目，上传GitHub。
  - 1月：期末复习，GPA冲刺。
- **大二下（2月-6月）**：
  - 2-3月：深化方向（AI: RAG/Agent；云: K8s部署）。中型项目启动。
  - 4月：考华为云HCIA（可选）。
  - 5月：考IBM Data Science。参加竞赛。
  - 6月：项目收尾，作品集整理。期末。
- **暑假（7-8月）**：
  - 7月：实习（AI/云公司），或独立大项目（上线应用）。
  - 8月：开源贡献+休息。预习大三内容。
---
#### 避坑指南
- **多而不精**：只深1方向，其他浅学。删减次要内容后，专注度更高。
- **忽略AI工具**：必须用Cursor/Claude优化代码，但独立设计逻辑。
- **证书堆砌**：只考2-3个，别超过。优先匹配方向。
- **只理论不实践**：每周至少1天动手项目，否则简历空洞。
- **GPA vs 项目**：平衡好，项目>纯分。目标：作品集有2-3个Demo + 证书。
- 目标：GPA 3.7++，每周学习时间：40-50小时（包括课程+自学）。
- 精力分配：70%核心课程+主方向，20%实践项目，10%证书+课外
- 初步构建作品集（Portfolio）。
**课外活动**：
- 贡献开源项目：如在GitHub上fork ML仓库。
- 证书：获取Coursera的Google IT Support或IBM Data Science证书。
- 关注趋势：跟踪量子计算进展（如IBM Quantum的开源工具），并尝试模拟实验。

## 大三：方向聚焦+项目深耕+实习冲刺

#### 核心目标：成为某一赛道“准专业人才”，积累高质量项目与实习经历，适配校招提前批需求。

#### 方向深耕（四选一，结合市场缺口）
- 大模型开发方向（年薪25-50万）：
	-  核心技术：PyTorch/TensorFlow框架、RAG检索增强、Lora微调、向量数据库（Milvus）；
	- 项目实战：开发垂直场景小模型（如医疗问答、代码助手），或参与开源大模型二次开发；
	- 能力要求：掌握Transformer架构、英文文献阅读能力（顶会论文精读）。
- 云原生架构方向（年薪20-40万）：
	- 核心技术：K8s编排、微服务架构（Spring Boot/Go Gin）、Serverless、ELK日志分析；
	- 项目实战：将单体应用拆分为微服务，通过K8s实现自动扩缩容；
	- 证书进阶：阿里云ACP、Google Cloud Associate认证（加分项）。
- 网络安全方向（人才缺口140万，年薪25-80万）：
	- 核心技术：渗透测试、漏洞挖掘、防火墙配置、隐私计算基础；
	- 项目实战：参与CTF竞赛，或搭建企业级安全防护模拟系统；
	- 能力要求：熟悉等保2.0、数据安全法合规要求。
- 嵌入式智能方向（需求年增15%）：
	- 核心技术：C++高性能编程、RTOS实时系统、车载/工业物联网开发；
	- 项目实战：开发智能监控设备、车载信息娱乐系统模块；
	- 适配场景：芯片、汽车、工业数字化等“抗中年危机”赛道。
- **后端/云原生**
    - 学习 Go / Java / Python 深入
    - 学习 Docker、Kubernetes、微服务架构
- **前端全栈**
    - 深入学习 React/Vue
    - TypeScript
- **人工智能/机器学习**
    - 线性代数进阶、概率统计
    - Python + TensorFlow/PyTorch
- **数据工程/大数据**
    - Hadoop、Spark、Kafka 入门
- **核心课程**：
  - 软件工程：设计模式、测试、敏捷方法。
  - 人工智能：深度学习、神经网络（用PyTorch）。
  - 选修方向：云计算（Docker/Kubernetes）、网络安全（渗透测试）、大数据（Hadoop/Spark）。
  - 其他：编译原理或计算机图形学。

- **技能培养**：
  - DevOps工具：CI/CD管道（如Jenkins）。
  - 团队协作：使用Agile工具（如Jira）。
  - 伦理知识：学习AI偏见和数据隐私（GDPR-like法规）。

---
#### 关键动作
- **技术选修课**：云计算、大数据、信息安全、嵌入式系统
- **大型团队项目**
- 准备 **实习简历 & 作品集**

- **项目/实践**：
  - 大型项目：团队开发App（如基于区块链的投票系统）或AI应用（如聊天机器人）。
  - 研究实习：申请学校实验室或企业R&D（如腾讯AI Lab）。
  - 暑假：全职实习（目标：知名科技公司如字节跳动或华为），积累1-2年经验。


**输出**

- 实习准备包（简历 + 技术题 + 项目演示）
- 实习：大二暑期/大三秋季争取大厂实习（如华为云、字节AI Lab、阿里安全），目标岗位与深耕方向一致；
- 算法刷题：LeetCode每周3-5题，重点突破数组、动态规划、图论（大厂技术面必考点）；
- 作品集打磨：整理2-3个高质量项目（含代码、文档、演示视频），突出“技术难点+解决方案”。

## 大四：就业冲刺+能力整合

#### 核心目标：适配校招/升学要求，完成“技术-岗位”精准匹配，顺利毕业。
##### 就业导向
- 简历优化：突出实习经历、项目成果、技术栈适配性（如“熟练使用Lora微调大模型”“掌握K8s集群部署”）；
- 校招备战：刷目标岗位高频面试题（计算机网络/操作系统/数据库核心考点），模拟技术面（重点：项目细节、技术选型逻辑）；
- 渠道拓展：重点关注大厂提前批、国央企研发岗（如国家电网、华为政企）、新一线城市科技公司（成都/武汉/杭州，落户政策宽松）。

- **就业导向**：到大四，目标有3-5个项目、2次实习、强Portfolio。当前市场青睐全栈开发+AI技能。
---
#### 升学导向
- 复试准备：聚焦目标方向前沿（如大模型压缩、云原生安全），准备1-2个科研项目（如参与导师的AI伦理、工业软件研发课题）；
- 毕设选题：优先选择“技术+行业”复合场景（如金融风控AI模型、医疗影像小模型），提升复试竞争力。
---
#### 毕设核心要求
- 避免“重复造轮子”，结合热门赛道做创新（如“基于RAG的智能办公助手优化”“K8s边缘计算节点调度算法改进”）；
- 体现“系统思维”，涵盖需求分析、架构设计、落地测试全流程。
**目标**

- 完成毕业设计 / 论文
- 全面准备职业发展

**准备内容**
- Algorithm 强化（经典题型 + 高频面试）
- 系统设计面试训练
- 行业方向深度准备（如前端、后端、AI）
**实践目标**
- 参加校招/社招
- 技术分享/技术博客发布

**输出**
- 毕业设计 Demo + 文档
- 面试题库 + 经验总结
---
#### 通用适配建议

1. 证书优先级：云厂商认证（阿里云ACP/华为云HCIP）> 安全合规认证（CISP）> 语言类证书（四六级，外企必备）；
2. 行业复合方向：可叠加“技术+场景”（如AI+金融、云原生+制造、安全+医疗），薪资比纯技术岗高40%；；
3. 软实力培养：团队协作（Git协作开发）、技术沟通（项目文档撰写）、伦理判断（AI隐私保护、数据合规）——这些是AI无法替代的核心能力。
4. 平衡求职与学习，目标就业薪资15-25万RMB/年（入门级）。如果读研，准备申请。
