## 基本的数据类型
- 主要知识点
	- %u无符号整数，%ld 长整数型，\n 换行 \0结束 \ddd八进制三位数 \xhh16进制
	- char  1个字节，short 2个字节，int 4个字节，float 4个字节，数据越界往高级转化
	- %02d，表示打印两个整数 用0来填充第一个位置
	- %.2f，浮点数保留两位有效数字
	- %%（%%）是转义字符，用于在 printf 中输出一个 % 符号                
			- \\n 是转义换行符，用于输出 \n 字符
		- \"Hello world!\"
	- 原码：数据的二进制表示
	- 反码：符号位不变，其他位数依次取反
	- 补码：反码+1，数值内存以补码存储，原码和补码可以相互转化
	- 符号位为首位，0表示整数，1表示整数
	- 整数的原码补码反码都相同
	- &补码两者有0为0
	- |补码两者有1为1
	- ^补码相异为1，相同为0
	- char类型本身为整型，存储ASCII值，char的符号并不确定
		- char arr[]="abcdef" 字符串的末尾隐藏着\0
		- char A[16] = { '1','2','3','4','5','6','7','8','9','A','B','C','D','E','F'};
		- ASCII值a65 A92
	- define 无类型 全局 文本交替
	- const 有类型 可取地址 作用域
	- 大小端存储，把高低位存储在不同地址，x86是小端存储，取决于硬件
- 应用场景
	- 不使用第三变量，交换变量值
		- a^a=0，0^a=a
	- 判断大小端
		- 取首地址强转为char解引用?1:0

## 结构体
- 主要知识点
	- 形象定义：设置结构体为一个档案带，得预设其中的所有内容，在主函数中定义结构体变量或者数组，也可以在结构体尾处定义
	- typedef
		- 简化基本类型->typedef unsigned long ulong
		- 简化结构体typedef struct {int x, y;} Point;Point p2;  // 直接用 Point，不用 struct
	- 结构体嵌套为在父结构体内部定义一个结构体变量， 在其外部定义一个子结构体档案袋
	- 结构体指针的定义成员名字，指针->成员来表示
	- 初定义可以指定顺序{.成员=XX，.成员=XX}
	- strcmp比较数组大小 相等为0，strcmp（a，b) 小于输出负数/大于输出正数,用于判断结构体内字符串是否对应
	- 字符串不用=赋值，得用strcpy（XX.XX，”XX“）；头文件为include<string.h>
	- 计算结构内存大小考虑对齐规则
		- 首个成员首项对齐，每个成员首地址必须在对应变量字节的倍数上，总内存大小为结构体内最大变量字节的整数倍，其他空开的内存编译器自动填充

# 文件操作
- 主要知识点
	- FILL * p=fopen () //判断指针是否为NULL，返回0/1来判断是否成功，用指针来放回值
	- fopen("filename","mode"):
	- fliename包含文件路径，自动转义字符
	- mode**(二进制文件：mp3，mp4，ppt，war (+b))**
		- ab(追加): 如果文件不存在，创建，存在即将文件指针移动至末尾
		- rb(读取):如果文件不存在，返回NULL，存在即返回文件句柄
		- wb(创建):如果文件不存在，创建，存在即清空所有内容
		- r+:
		- w+:存在即清空文件，不存在则创建，可读可写
		- a+:存在即不清空文件，不存在则失败，用于读写
	- fwirte(内容存放位置,写入大小,写入次数,文件句柄);
	- fread(内容存放位置，读取大小,读取次数,文件句柄)
	- 存放位置为字符型数组
	- 中文不好计算字节大小
		- sizeof()  读取字符串的'/0'
		- strlen() 一般使用这个
	**函数返回值都是次数，读取写入量为大小* 次数，即规定大小为1，次数为总量**
	- fseek(文件柄权,偏移量,偏移位置)
		- 偏移量：正左负右
		- 偏移位置：SEEK_SET->0/SEEK_END->2/SEEK_CUR->1
	- ftell(* p)   //计算文件指针偏移量
	- fflush(* p)   //清空文件缓冲区
	- fclose(* p)    //关闭文件，成功返回0，失败返回非0
	- char * fgets(char * s, int n, FILE * fp) //最多读取n-1个字符
	- int fputs(const char * s, FILE * fp);  //若出现写入错误，则返回EOF，否则返回一个非负数。
	- 宏定义
	_ int _ 需要传入地址值
	_  out _ 函数参数做返回值
	- 文本显示为：GB2312（简体中文），UTF-8
