# 黄帝内经饮食调理身体

> 目标：参考《黄帝内经》的养生思想，通过饮食调理脾胃，帮助增加气血、减少湿气、提高精力，让身体回到更平衡、稳定、有节律的状态。

这份笔记不是药方，也不是疾病治疗方案。它更适合用于日常饮食调整：吃得有节制、顺应四季、照顾脾胃、不过寒不过热、不过饱不过偏。

## 一、核心思想

《黄帝内经》谈饮食，不只是讲“吃什么补什么”，而是强调身体整体的平衡。

可以抓住五个关键词：

1. 阴阳平衡：寒热、动静、虚实都要避免走极端。
2. 饮食有节：吃饭要有规律，不暴饮暴食。
3. 五味调和：酸、苦、甘、辛、咸都可以吃，但不可偏嗜。
4. 顺应四时：春夏秋冬饮食侧重点不同。
5. 脾胃为本：消化吸收能力好，身体才有气血来源。

本文件的具体调理方向：

- 增加气血：重点是养好脾胃、保证优质蛋白、主食不过度减少、睡眠规律。
- 减少湿气：重点是少甜、少油、少冰、少酒，减少身体消化负担。
- 提高精力：重点是三餐稳定、血糖稳定、不过饱、不过度刺激。

## 二、饮食有节

饮食调理的第一步不是找补品，而是恢复节律。

### 基本原则

- 三餐尽量固定时间。
- 每餐吃到七八分饱。
- 少吃过冷、过烫、过油、过甜、过咸的食物。
- 晚餐不要太晚、太撑、太油。
- 情绪很差、很急、很累时，先慢下来再吃饭。

### 判断吃得是否合适

吃完饭后，如果出现这些情况，通常说明饮食需要调整：

- 饭后明显困倦、头昏。
- 胃胀、反酸、嗳气。
- 大便长期不成形或长期干结。
- 口干口苦、口臭明显。
- 体重短期波动很大。
- 睡前胃里仍然很撑。

## 三、五味调和

《黄帝内经》认为五味和五脏有对应关系，但重点不是机械对应，而是“不要偏食偏嗜”。

| 味道 | 常见食物 | 日常作用倾向 | 过度偏嗜可能的问题 |
|---|---|---|---|
| 酸 | 山楂、乌梅、醋、柠檬、酸味水果 | 收敛、生津、帮助开胃 | 太酸可能影响胃部舒适，也可能让人感觉收得太紧 |
| 苦 | 苦瓜、茶、莲子心、部分绿叶菜 | 清爽、降腻、帮助减少厚重感 | 太苦寒可能伤胃口，胃弱者不宜过量 |
| 甘 | 米饭、红薯、南瓜、山药、枣、蜂蜜 | 补益、缓和、提供能量 | 太甜容易生湿、生痰、困倦、长胖 |
| 辛 | 姜、葱、蒜、辣椒、胡椒 | 发散、温通、开胃 | 太辛辣容易上火、口干、便秘、胃痛 |
| 咸 | 海带、紫菜、盐、酱油、咸菜 | 软坚、入肾、补充咸味需求 | 太咸容易水肿、口渴，也不利于血压管理 |

### 实用做法

- 一餐里尽量有主食、蛋白质、蔬菜，不要只吃一种味道。
- 口味重的人，先从少放盐、少放辣、少喝甜饮料开始。
- 想吃酸、辣、甜时，不必完全压制，但要看身体反应。
- 连续几天吃得厚重油腻，可以用清淡饮食调回来。

## 四、三大目标调理方案

## 1. 增加气血

《黄帝内经》的思路里，气血不是靠猛补出来的，而是靠脾胃把食物转化出来。想气血更足，先要让身体能消化、能吸收、能休息。

适合关注的人：

- 容易疲劳，脸色暗淡或偏白。
- 运动后恢复慢。
- 手脚容易冷。
- 睡眠不足后明显没精神。
- 吃得少、胃口差，或长期节食。

饮食重点：

- 每餐保留主食，不要长期低碳水。
- 每天有足量蛋白质，如鸡蛋、鱼、鸡肉、牛肉、豆腐。
- 多吃温和健脾食物，如山药、南瓜、小米、莲子、红枣。
- 少吃冰饮、生冷沙拉、过多水果代餐。
- 晚餐别太撑，避免影响睡眠和脾胃修复。

常用食材：

| 类别 | 推荐食材 | 用法 |
|---|---|---|
| 主食 | 大米、小米、燕麦、红薯、山药、南瓜 | 作为气血来源，不要完全断主食 |
| 蛋白质 | 鸡蛋、鱼、鸡肉、牛肉、虾、豆腐 | 每餐至少有一种 |
| 温和补养 | 红枣、桂圆、莲子、黑芝麻、枸杞 | 少量搭配，不要当零食大量吃 |
| 蔬菜 | 胡萝卜、菠菜、菌菇、西兰花、白菜 | 熟吃为主，帮助消化吸收 |

推荐组合：

- 小米山药粥 + 鸡蛋 + 熟青菜。
- 牛肉胡萝卜汤 + 米饭 + 菌菇。
- 鸡汤山药 + 杂粮饭 + 炒菠菜。
- 鱼肉豆腐汤 + 米饭 + 西兰花。

避坑：

- 只吃红枣、枸杞、阿胶类补品，却三餐混乱，效果通常不好。
- 气血不足但湿气重的人，不要一上来就大量甜腻滋补。
- 熬夜会直接消耗精力，再好的饮食也很难补回来。

## 2. 减少湿气

湿气重，常常和脾胃运化能力弱、饮食太甜太油太冷、久坐少动有关。饮食调理的重点是减负，而不是只喝祛湿茶。

适合关注的人：

- 身体沉重，早起困。
- 舌苔厚腻，口中黏。
- 大便黏滞、不爽或容易不成形。
- 脸油、痘痘、腹胀。
- 饭后特别困。

饮食重点：

- 少甜食、奶茶、蛋糕、含糖饮料。
- 少油炸、烧烤、肥肉、重口味夜宵。
- 少冰饮、冰水果、长期生冷饮食。
- 晚餐清淡，饭后散步 10-20 分钟。
- 主食适量，不要一边高油高糖一边完全不吃饭。

常用食材：

| 类别 | 推荐食材 | 用法 |
|---|---|---|
| 利湿清爽 | 冬瓜、薏米、赤小豆、白萝卜、玉米须 | 煮汤或煮水，配合清淡饮食 |
| 健脾 | 山药、莲子、芡实、小米、南瓜 | 适合湿气重又脾胃弱的人 |
| 蔬菜 | 绿叶菜、菌菇、黄瓜、丝瓜 | 熟食为主，少油 |
| 蛋白质 | 鱼、虾、鸡肉、豆腐、鸡蛋 | 清蒸、炖煮、少煎炸 |

推荐组合：

- 冬瓜虾仁汤 + 米饭 + 清炒青菜。
- 赤小豆薏米水 + 正常三餐，不要代替吃饭。
- 山药莲子粥 + 鸡蛋 + 熟蔬菜。
- 白萝卜牛肉汤 + 少量主食。

避坑：

- 只喝祛湿茶，但继续奶茶、夜宵、油炸，湿气很难降。
- 薏米偏清利，脾胃很虚、容易腹泻的人不要大量长期喝。
- 湿气重不等于不能吃肉，关键是少油、适量、易消化。

## 3. 提高精力

精力不是靠咖啡和刺激性食物硬顶出来的。稳定精力的关键是：规律进食、血糖稳定、脾胃负担适中、睡眠能恢复。

适合关注的人：

- 上午精神差，下午犯困。
- 饭后脑子钝、想睡。
- 靠咖啡、奶茶、甜食续命。
- 训练或学习时容易没劲。
- 睡醒仍然累。

饮食重点：

- 早餐要有蛋白质，不要只吃甜面包或只喝饮料。
- 午餐别太油太撑，避免饭后困。
- 训练或学习前可以吃少量主食和蛋白质。
- 咖啡不要空腹喝，也不要靠咖啡代替睡眠。
- 晚上少吃重口味夜宵，保护睡眠质量。

常用食材：

| 时间 | 推荐吃法 | 目的 |
|---|---|---|
| 早餐 | 鸡蛋 + 粥/燕麦/全麦主食 + 一点水果 | 稳定上午精力 |
| 午餐 | 米饭 + 蛋白质 + 两份蔬菜 | 防止下午崩盘 |
| 下午 | 坚果少量、酸奶、香蕉、鸡蛋 | 替代甜饮和零食 |
| 晚餐 | 热汤 + 蛋白质 + 熟蔬菜 + 少量主食 | 不撑胃，帮助睡眠 |

推荐组合：

- 早餐：燕麦 + 鸡蛋 + 香蕉半根。
- 午餐：米饭 + 鸡肉/鱼肉 + 两份蔬菜。
- 下午：无糖酸奶或坚果少量。
- 晚餐：豆腐鱼汤 + 青菜 + 少量米饭。

避坑：

- 早餐只喝咖啡，容易上午虚、午后暴食。
- 午餐太油太撑，下午精力会被消化拖走。
- 晚上吃太饱，第二天精神也会受影响。

## 五、顺应四季饮食

四季饮食的重点是顺着气候变化，而不是硬套补法。

## 春季

春天阳气生发，饮食宜清爽、舒展，避免太油腻。

适合：

- 绿叶菜、豆芽、韭菜、春笋。
- 鸡蛋、鱼、豆制品等轻负担蛋白质。
- 米饭、小米、山药、红薯等温和主食。

少吃：

- 过多油炸、烧烤、甜食。
- 过量辛辣刺激。

调理重点：

- 帮助身体从冬天的厚重状态中舒展开。
- 情绪容易郁闷、胸口闷、食欲差时，饮食要清淡一点，配合散步。

## 夏季

夏天炎热，容易出汗、口渴、困倦。饮食宜清淡、补水、不过度寒凉。

适合：

- 绿豆汤、冬瓜、丝瓜、黄瓜、西红柿。
- 粥、面、米饭等容易消化的主食。
- 鱼、虾、鸡肉、豆腐等清淡蛋白质。

少吃：

- 冰饮、冰西瓜、冷奶茶一次吃太多。
- 重油重辣火锅、烧烤、夜宵。

调理重点：

- 出汗多时注意补水和盐分。
- 胃口差时可以少量多餐，但不要长期只喝冷饮吃水果。

## 秋季

秋天偏燥，饮食宜润燥、生津、少辛辣。

适合：

- 梨、银耳、百合、莲藕、山药。
- 芝麻、核桃、蜂蜜少量使用。
- 白萝卜、南瓜、菌菇类。

少吃：

- 过多辣椒、烧烤、酒。
- 过度煎炸、干硬食物。

调理重点：

- 口干、咽干、皮肤干、大便干时，少辛辣，多喝温水，多吃润燥食物。

## 冬季

冬天寒冷，饮食宜温养、藏精、不要过度生冷。

适合：

- 羊肉、牛肉、鸡肉、鱼肉，按体质适量。
- 山药、栗子、黑芝麻、核桃、黑豆。
- 萝卜、白菜、土豆、胡萝卜。
- 姜枣茶、热粥、热汤。

少吃：

- 大量冷饮、生冷水果。
- 天天大补、顿顿厚味。

调理重点：

- 怕冷、手脚凉的人，少吃生冷，多吃热饭热菜。
- 容易上火、口干、便秘的人，即使冬天也不要盲目温补。

## 六、根据常见状态调整

以下是日常调理思路，不用于替代诊断。

## 1. 脾胃虚弱型

常见表现：

- 胃口差，饭后胀。
- 大便偏软或不成形。
- 容易疲劳、困倦。
- 吃冷的、油的就不舒服。

饮食调整：

- 多吃温热熟食，少吃生冷。
- 主食可选米饭、小米粥、山药、南瓜。
- 蛋白质选择鸡蛋、鱼、鸡肉、豆腐，少油烹调。
- 少喝冰饮、奶茶、酒。
- 晚餐简单一点，给脾胃留恢复空间。

可参考组合：

- 小米粥 + 鸡蛋 + 炒青菜。
- 山药排骨汤 + 米饭 + 清炒蔬菜。
- 南瓜粥 + 豆腐 + 少量瘦肉。

## 2. 湿重困倦型

常见表现：

- 身体沉重，容易困。
- 舌苔厚腻。
- 大便黏滞不爽。
- 脸油、痘痘、口中黏腻。

饮食调整：

- 减少甜食、油炸、奶茶、酒。
- 主食不过量，避免饭后立刻躺下。
- 多吃冬瓜、薏米、赤小豆、菌菇、绿叶菜。
- 烹调方式以蒸、煮、炖为主。
- 配合每天散步或轻微出汗。

可参考组合：

- 赤小豆薏米水，不要当作唯一饮食。
- 冬瓜汤 + 米饭 + 清蒸鱼。
- 菌菇青菜汤 + 鸡蛋 + 少量主食。

## 3. 阴虚燥热型

常见表现：

- 口干、咽干。
- 手心脚心热。
- 睡眠浅、容易烦。
- 大便偏干。

饮食调整：

- 少吃辛辣、烧烤、酒。
- 多吃梨、银耳、百合、莲藕、芝麻、豆腐。
- 蛋白质选择鱼、蛋、豆制品，少煎炸。
- 晚上避免重口味夜宵。

可参考组合：

- 银耳百合羹，少放糖。
- 莲藕排骨汤 + 米饭 + 青菜。
- 豆腐鱼汤 + 蔬菜 + 少量主食。

## 4. 阳虚怕冷型

常见表现：

- 手脚凉，怕冷。
- 喜欢热饮。
- 吃冷食后腹泻或胃痛。
- 精神偏低，容易疲劳。

饮食调整：

- 少吃冰饮、生冷沙拉、寒凉水果。
- 多吃热饭热汤。
- 可适量用姜、葱、胡椒暖胃。
- 选择羊肉、牛肉、鸡肉、山药、栗子等温养食物。
- 运动量不要太猛，但要保持规律活动。

可参考组合：

- 姜丝鸡汤 + 米饭 + 青菜。
- 羊肉萝卜汤，体热或上火时少吃。
- 山药牛肉粥。

## 5. 肝郁气滞型

常见表现：

- 情绪憋闷、容易叹气。
- 胸胁胀、胃口受情绪影响。
- 压力大时胃胀或想吃甜食。
- 睡前脑子停不下来。

饮食调整：

- 少用暴食、甜食、酒精缓解压力。
- 饮食清爽，避免长期厚重油腻。
- 可用陈皮、玫瑰花、佛手、山楂等做温和茶饮。
- 多吃绿叶菜、柑橘类水果、萝卜、豆制品。
- 饭后散步比立刻刷手机更有用。

可参考组合：

- 陈皮山楂水，胃酸明显者少用山楂。
- 绿叶菜 + 鱼肉 + 米饭。
- 白萝卜汤 + 豆腐 + 少量主食。

## 七、一天饮食模板

## 温和日常版

早餐：

- 小米粥或燕麦粥。
- 鸡蛋 1 个。
- 少量坚果或一份水果。

午餐：

- 米饭或杂粮饭。
- 鱼、鸡肉、牛肉、豆腐任选一种。
- 两份蔬菜，一份深色叶菜。

晚餐：

- 热汤或粥。
- 清淡蛋白质。
- 熟蔬菜。
- 不吃太撑。

## 湿重清爽版

早餐：

- 小米粥或玉米。
- 鸡蛋。
- 温水。

午餐：

- 米饭减量。
- 冬瓜、菌菇、绿叶菜。
- 清蒸鱼或鸡胸肉。

晚餐：

- 蔬菜豆腐汤。
- 少量主食。
- 不吃甜点和夜宵。

## 怕冷温养版

早餐：

- 热粥。
- 鸡蛋。
- 姜枣水少量。

午餐：

- 米饭。
- 牛肉、鸡肉或鱼。
- 熟蔬菜。

晚餐：

- 山药汤、萝卜汤或鸡汤。
- 少量主食。
- 避免冰饮和生冷水果。

## 八、饮食禁忌与边界

以下情况不要只靠饮食调理：

- 持续发热、明显疼痛、呕吐腹泻不止。
- 体重短期明显下降。
- 便血、黑便、严重胃痛。
- 长期失眠、心悸、胸痛。
- 高血压、糖尿病、肾病、痛风等慢病需要特殊饮食管理。
- 正在服药、备孕、孕期、哺乳期。

如果已有疾病或正在用药，饮食调整应先问医生或营养师，尤其不要自行长期使用药食同源材料或补品。

## 九、每周复盘

每周用 5 分钟记录：

| 项目 | 记录 |
|---|---|
| 胃口 | 好 / 一般 / 差 |
| 饭后状态 | 轻松 / 胀 / 困 / 反酸 |
| 大便 | 正常 / 偏稀 / 偏干 / 黏滞 |
| 睡眠 | 好 / 一般 / 差 |
| 口渴口干 | 无 / 轻微 / 明显 |
| 怕冷或怕热 | 无 / 怕冷 / 怕热 |
| 本周最该调整的一件事 | 例如少冰饮、少夜宵、少甜食、早点吃晚饭 |

## 十、最低执行版本

如果不想想太复杂，只执行这 6 条：

1. 三餐规律。
2. 七八分饱。
3. 少冰饮、少夜宵、少油炸、少甜饮。
4. 每餐有主食、蛋白质和蔬菜。
5. 春夏清爽，秋冬温润。
6. 看饭后状态和大便情况来调整饮食。

饮食调理的关键不是追求神奇食物，而是让身体长期处在更稳定、更少负担的节律里。
