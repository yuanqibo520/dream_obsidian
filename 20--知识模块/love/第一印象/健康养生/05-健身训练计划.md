# 健身训练计划

> 目标：每周有明确安排，能照着练、能记录、能根据疲劳调整。适合塑形、增肌、减脂和提升体能。

## 一、总原则

- 每周力量训练 3-4 次。
- 每周有氧 2-3 次。
- 每天步数尽量 7000-10000 步。
- 每次训练：热身 8-12 分钟，正式训练 40-60 分钟，拉伸 5-10 分钟。
- 每组结束时保留 1-3 次力竭余量，不追求每组练到崩。
- 睡眠尽量 7-9 小时。睡前可用 4-7-8 呼吸法放松：吸气 4 秒、屏息 7 秒、呼气 8 秒，循环 3-5 轮。

## 二、12 周进阶安排

### 第 1-4 周：建立动作和习惯

- 每周 3 练。
- 每个动作 2-3 组。
- 每组 10-15 次。
- 重点：动作标准、找合适重量、不要追重量。

推荐周安排：

| 星期 | 内容 |
|---|---|
| 周一 | 全身训练 A |
| 周二 | 休息或散步 |
| 周三 | 全身训练 B |
| 周四 | 休息 |
| 周五 | 全身训练 A |
| 周六 | 轻有氧 30 分钟 |
| 周日 | 休息和复盘 |

### 第 5-8 周：稳定增肌塑形

- 每周 4 练。
- 每个动作 3-4 组。
- 每组 8-12 次。
- 重点：逐步加重量，记录训练数据。

推荐周安排：

| 星期 | 内容 |
|---|---|
| 周一 | 上肢推 |
| 周二 | 下肢 |
| 周三 | 低强度有氧和拉伸 |
| 周四 | 上肢拉 |
| 周五 | 下肢 + 核心 |
| 周六 | 可选有氧或运动日 |
| 周日 | 休息和复盘 |

### 第 9-12 周：提高强度和线条

- 每周 4 次力量 + 2 次有氧。
- 复合动作 4 组 x 6-10 次。
- 孤立动作 3 组 x 12-15 次。
- 重点：保持动作质量，增加训练密度。

## 三、每周直接执行版

### 周一：胸、肩、三头

1. 跑步机快走或椭圆机 8 分钟。
2. 器械推胸 4 组 x 8-12 次。
3. 哑铃卧推 3 组 x 8-12 次。
4. 坐姿肩推 3 组 x 8-12 次。
5. 哑铃侧平举 3 组 x 12-15 次。
6. 绳索下压 3 组 x 12-15 次。
7. 平板支撑 3 组 x 30-60 秒，双腿略打开。

### 周二：腿、臀

1. 动态热身：髋关节绕环、徒手深蹲、弓步各 1-2 组。
2. 腿举机 4 组 x 10-12 次。
3. 高脚杯深蹲 3 组 x 10-12 次。
4. 罗马尼亚硬拉 3 组 x 8-12 次。
5. 臀桥或臀推 4 组 x 10-15 次。
6. 坐姿腿弯举 3 组 x 12-15 次。
7. 小腿提踵 3 组 x 12-20 次。

### 周三：低强度有氧和拉伸

- 快走、骑车、椭圆机任选 30-45 分钟。
- 强度：能说话但不能唱歌。
- 拉伸：胸、背、髋屈肌、腘绳肌、小腿，每个位置 30-45 秒。

### 周四：背、二头、后束

1. 划船机或椭圆机 8 分钟。
2. 高位下拉 4 组 x 8-12 次。
3. 坐姿划船 4 组 x 8-12 次。
4. 单臂哑铃划船 3 组 x 每侧 10-12 次。
5. 面拉 3 组 x 12-15 次。
6. 哑铃弯举 3 组 x 10-15 次。
7. 死虫动作 3 组 x 每侧 8-12 次。

### 周五：腿、核心、体态

1. 慢跑或快走 5-10 分钟。
2. 史密斯深蹲或徒手深蹲 4 组 x 8-12 次。
3. 保加利亚分腿蹲 3 组 x 每侧 8-10 次。
4. 腿屈伸 3 组 x 12-15 次。
5. 髋外展机 3 组 x 12-20 次。
6. 小鸟喝水 3 组 x 每侧 8-12 次。
7. 侧桥 3 组 x 每侧 20-45 秒。

### 周六：可选有氧或运动日

- 减脂优先：中低强度有氧 40-60 分钟。
- 增肌优先：散步 7000-10000 步，不做高强度有氧。
- 疲劳明显：休息、泡脚、拉伸。

### 周日：休息和复盘

- 记录体重、腰围、训练重量、睡眠、食欲。
- 只看 7 天平均体重，不被单日波动影响。

## 四、全身训练 A/B

如果一开始不想每周 4 练，可以用全身训练。

### 全身训练 A

1. 腿举机 3 组 x 10-12 次。
2. 器械推胸 3 组 x 10-12 次。
3. 高位下拉 3 组 x 10-12 次。
4. 臀桥 3 组 x 12-15 次。
5. 平板支撑 3 组 x 30-45 秒。

### 全身训练 B

1. 高脚杯深蹲 3 组 x 10-12 次。
2. 坐姿划船 3 组 x 10-12 次。
3. 哑铃卧推 3 组 x 8-12 次。
4. 罗马尼亚硬拉 3 组 x 8-12 次。
5. 死虫动作 3 组 x 每侧 8-12 次。

## 五、加重量规则

- 某动作目标是 8-12 次时，如果所有组都能完成 12 次且动作稳定，下次加重量。
- 上肢动作每次加 1-2.5 kg 更稳。
- 下肢动作每次加 2.5-5 kg 更稳。
- 如果加重量后动作变形，先退回原重量。

## 六、疲劳调整

- 睡眠低于 6 小时：当天重量降 10-20%。
- 连续 3 天疲劳评分 7 分以上：减少 1-2 个动作或少做 1 组。
- 关节痛：停止对应动作，换低冲击版本。
- 胃肠不舒服：训练前别吃太油太撑。
- 晚间睡眠差：晚上不要做高强度训练。

## 七、最低执行版本

一周很忙时只做下面这些：

- 力量训练 3 次：推胸、下拉、腿举、臀桥、平板支撑，每个 3 组。
- 有氧 2 次：快走或椭圆机 30 分钟。
- 每天尽量散步。
- 睡前 10 分钟放下手机，做 4-7-8 呼吸。

## 九、二手健身卡检查

如果考虑买二手健身卡，先确认：

- 健身房离家或学校足够近。
- 是否能正常转卡，转卡费谁付。
- 剩余有效期和可用门店。
- 器材是否包含跑步机、椭圆机、推胸、下拉、划船、腿举、史密斯、哑铃区。
- 高峰期是否拥挤到影响训练。
