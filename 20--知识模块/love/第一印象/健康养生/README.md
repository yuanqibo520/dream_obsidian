# 健康养生文件夹说明

本文件夹用于整理个人健康、基础医药、体态改善、健身训练与饮食相关的知识笔记。当前文件均为 Markdown 文档，适合在 Obsidian 或其他 Markdown 编辑器中阅读。

## 建议阅读顺序

1. `01-基本必备医药知识.md`：先了解常见不适、常备药和基础用药边界。
2. `体态.md`：建立体态安全边界、自测方法和每日基础训练。
3. `04-健身器材使用与身体部位.md`：熟悉健身房器材、动作对应的身体部位和安全要点。
4. `05-健身训练计划.md`：根据阶段安排每周训练，并记录进度。
5. `06-健身饮食计划.md`：配合训练调整蛋白质、碳水、脂肪、水分和补剂策略。

## 文件功能一览

| 文件 | 主要功能 |
|---|---|
| `01-基本必备医药知识.md` | 汇总基础医药常识，覆盖感冒发烧止痛、肠胃不适、咳嗽咽喉、过敏皮肤、外伤应急、老人和慢病常备等场景；重点是常备用药、处理提醒和用药安全边界。 |
| `体态.md` | 体态改善教程，包含安全边界、体态底层逻辑、5 分钟自测、日常坐站走原则、每日 12 分钟基础训练、4 周进阶安排、1 分钟重置和常见问题。 |
| `04-健身器材使用与身体部位.md` | 健身器材和动作说明，整理器材/动作与主要训练部位的对应关系，并按胸、背、肩、手臂、腿臀、核心、有氧器材等分类说明使用方式和安全检查。 |
| `05-健身训练计划.md` | 健身训练执行方案，包含训练总原则、12 周进阶安排、每周训练日程、全身训练 A/B、加重量规则、疲劳调整、最低执行版本、训练记录模板和二手健身卡检查。 |
| `06-健身饮食计划.md` | 健身饮食与营养方案，覆盖核心饮食原则、盘子法、蛋白质/碳水/脂肪选择、水和电解质、一天饮食模板、减脂/增肌调整、补剂建议、养生配方和每周复盘。 |

## 使用提示

- 医药相关内容只作为日常知识整理，出现严重症状、持续不适、慢病用药冲突或不确定情况时，应优先咨询医生或药师。
- 健身和体态内容建议循序渐进执行；疼痛、麻木、刺痛或异常疲劳时应降低强度或暂停训练。
- 训练计划、饮食计划和体态训练可以组合使用：先保证安全和动作质量，再逐步增加训练量和饮食精度。
