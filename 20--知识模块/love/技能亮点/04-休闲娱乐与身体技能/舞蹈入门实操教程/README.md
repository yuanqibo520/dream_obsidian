# 舞蹈入门实操教程

这套教程适合零基础到初级水平，目标不是先追求复杂动作，而是先把身体控制、节奏感、动作质感和现场表现练起来。

## 学习顺序

1. 先看《01-零基础舞蹈入门主教程.md》
2. 再按《02-14天训练计划.md》每天练
3. 练习中随时查《03-经验类实操技巧.md》

## 你要先建立的三个观念

1. 跳舞不是背动作，而是让身体跟着音乐做有控制的表达。
2. 初学者最重要的是节奏、重心、身体放松，不是动作数量。
3. 录视频复盘比盲练有效得多，每次只改一个问题。

## 每次练习的基本流程

1. 热身 5-8 分钟
2. 节奏训练 5 分钟
3. 基础动作 15-20 分钟
4. 小组合串联 10-15 分钟
5. 录视频复盘 5 分钟

## 安全提醒

- 膝盖、脚踝、腰部不舒服时不要硬跳。
- 地面太滑或太涩都不适合练习。
- 初学阶段少做大幅度甩头、下腰、跪地和爆发跳跃。
