# 拍摄前中后SOP

## 拍前：10分钟准备

### 1. 明确主题

先问自己：这次要拍什么感觉？

- 干净清爽
- 傍晚氛围
- 夜景电影感
- 甜一点
- 冷一点
- 松弛街拍
- 日常记录

主题越明确，现场越不容易乱。

### 2. 准备参考图

准备 3-6 张就够，不要太多。

每张参考图只看四件事：

- 光从哪里来？
- 人站在哪里？
- 背景有什么？
- 动作怎么做？

### 3. 检查装备

- 手机或相机电量。
- 镜头是否干净。
- 是否需要支架。
- 是否需要伞、外套、饮料、包、花等道具。
- 夜景是否需要补光或找橱窗光。

### 4. 现场踩点

到现场后先找：

- 最干净的背景。
- 最好看的光。
- 最安全的位置。
- 可以休息和缓冲的位置。
- 备选点位。

## 拍中：按顺序推进

### 第一步：环境空镜

先拍 5-10 张环境，不急着拍人。

作用：

- 找构图。
- 找光线。
- 让自己进入状态。
- 让被拍的人放松。

### 第二步：远景全身

先拍远一点，让压力小一点。

注意：

- 脚靠近画面底部。
- 头顶留少量空间。
- 手机不要从太高处往下拍。
- 背景线条不要切头。

### 第三步：半身和侧脸

用 2x 或 3x 更容易压缩背景，也更显脸小。

注意：

- 身体侧 30 度。
- 脸转向光源。
- 肩膀放松。
- 手里最好有东西可拿。

### 第四步：特写和情绪

不要一上来就怼脸拍。等状态松了再拍。

可拍内容：

- 眼神。
- 头发。
- 手部细节。
- 低头笑。
- 回头一瞬间。

### 第五步：合照或氛围照

最后拍合照，因为这时最自然。

优先顺序：

1. 背影合照。
2. 影子合照。
3. 侧面合照。
4. 正面合照。

## 拍后：选片和复盘

### 1. 先粗选

第一遍只删掉：

- 糊掉的。
- 表情明显失败的。
- 闭眼的。
- 构图严重歪的。
- 背景明显穿帮的。

### 2. 再精选

第二遍按用途选：

- 头像图。
- 朋友圈图。
- 氛围图。
- 细节图。
- 记录图。

### 3. 调色原则

- 先调曝光，再调色彩。
- 人脸肤色优先。
- 夜景不要修得像白天。
- 不要为了滤镜牺牲皮肤质感。
- 一组图保持色温统一。

### 4. 复盘

用 [[06-作品复盘模板]] 记录：

- 哪张最好，为什么？
- 哪张最失败，为什么？
- 下次只改哪一个问题？

## 保底拍摄公式

当现场不知道怎么拍时，按这个公式：

1. 找一块干净背景。
2. 让人站到阴影边缘或窗边光。
3. 用 2x 拍半身。
4. 身体侧 30 度，脸看向光。
5. 手拿饮料或整理头发。
6. 连拍 10 张。
