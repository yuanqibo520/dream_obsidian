## 后续维护规则

1. 新增技能资料先判断用途，再放入四个一级目录之一。
2. 单一技能有 3 篇以上资料时，单独建文件夹，并保留一个总览或 README。
3. 实操类资料优先包含：学习目标、训练步骤、检查标准、复盘模板。
4. 变现类资料优先包含：适合人群、进入门槛、路径拆解、关键成果变量、风险点。
5. 空文件不要长期保留，至少补一个“待补充提纲”。
