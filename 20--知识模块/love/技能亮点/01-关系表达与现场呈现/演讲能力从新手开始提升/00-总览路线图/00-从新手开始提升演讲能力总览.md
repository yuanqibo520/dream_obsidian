# 从新手开始提升演讲能力总览

## 核心目标

把“能开口”练成“讲得清楚、有感染力、能控场”。新手阶段不要一上来追求震撼全场，先做到三件事：

1. 不怯场：能在别人面前完整讲完。
2. 有结构：别人听完知道你想表达什么。
3. 有状态：声音、眼神、姿态让人愿意继续听。

## 能力模块

| 模块 | 解决的问题 | 新手优先级 |
|---|---|---|
| 心理建设与破冰 | 紧张、怕出错、不敢开口 | 高 |
| 内容组织与逻辑表达 | 不知道讲什么、讲乱了 | 高 |
| 声音语言与节奏 | 声音小、语速乱、口头禅多 | 高 |
| 肢体台风与镜头感 | 站姿僵硬、眼神躲闪 | 中 |
| 开场结尾与故事力 | 开头尴尬、结尾没力量 | 中 |
| PPT视觉辅助 | PPT堆字、念稿 | 中 |
| 互动控场与临场应变 | 被打断、冷场、答不上来 | 进阶 |
| 复盘反馈与长期成长 | 练了但不知道有没有进步 | 高 |

## 新手四阶段路线

### 第 1 阶段：敢讲

时间：第 1 周

重点：每天 1 分钟即兴表达，先降低开口成本。

达标：能面对手机镜头连续讲 60 秒，不中断、不重录。

### 第 2 阶段：讲清楚

时间：第 2 周

重点：用“结论-理由-例子-回扣”组织观点。

达标：3 分钟表达里，听众能复述出你的核心观点。

### 第 3 阶段：讲好听

时间：第 3 周

重点：声音、停顿、重音、眼神和手势。

达标：录音回听时，语速稳定，口头禅明显减少。

### 第 4 阶段：能上场

时间：第 4 周

重点：完成一次 5 分钟正式演讲，并做复盘。

达标：有完整稿件、简洁PPT、一次录像、一次复盘。

## 每次练习的最小闭环

1. 选一个小主题。
2. 写 3 个要点。
3. 讲 1 到 3 分钟。
4. 录音或录像。
5. 只改一个问题。

## 新手不要踩的坑

- 不要一开始背长稿，容易更紧张。
- 不要只看技巧，不开口练习。
- 不要每次都想表现完美，先完成再优化。
- 不要把PPT当演讲稿，PPT只服务听众理解。
