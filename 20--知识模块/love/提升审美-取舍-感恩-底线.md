# 提升审美，懂取舍，知感恩，守底线

这四件事其实是一条线：看得见好东西，分得清轻重，记得别人给过什么，也知道自己不能丢什么。

## 提升审美

多看高质量的东西，少被低质量刺激喂养。

可以看建筑、电影、摄影、书法、绘画、服装、产品设计，也可以看人的表达方式和生活秩序。审美不是“喜欢高级”，而是慢慢能分辨：什么是干净、克制、准确、真诚，什么只是热闹、堆砌、炫耀。

一个简单练习：

1. 每天选一件你觉得好的东西。
2. 问自己：它好在哪里？
3. 问自己：它少做了什么？
4. 问自己：我能不能学一点到自己的生活里？

## 懂取舍

取舍的核心不是“我不要什么”，而是“我最在乎什么”。

人痛苦很多时候不是因为没得选，而是想什么都要：要自由也要安全，要体面也要轻松，要关系也要绝对自我。

可以常问自己：

1. 这件事五年后还重要吗？
2. 它是在消耗我，还是在建设我？
3. 我现在的选择，是出于恐惧、虚荣，还是价值？

## 知感恩

感恩不是低姿态，也不是欠人情，而是心里有数。

知道有人帮过你、托举过你、忍让过你，知道今天很多东西不是凭空来的。

最实在的感恩，是别把别人的好当理所当然：

1. 及时回应。
2. 认真记住。
3. 能力到了就回馈。
4. 暂时回馈不了，也别辜负。

## 守底线

底线不是嘴上说“我有原则”，而是在诱惑、压力、情绪、利益面前还能不越界。

可以提前写清楚自己的几条底线：

1. 不欺骗真心待我的人。
2. 不靠伤害别人换好处。
3. 不在愤怒时做不可逆的事。
4. 不为了合群出卖尊严。
5. 不拿别人的信任开玩笑。

底线越早想清楚，关键时刻越不容易被情绪带走。

## 每日四问

每天睡前写四句话：

1. 今天我看见了什么美？
2. 今天我舍弃了什么不重要的东西？
3. 今天谁值得我感谢？
4. 今天我有没有守住自己的边界？

坚持一段时间，人会慢慢变得清爽。不是突然“高级”了，而是心里开始有秩序了。
