常用命令
- ls  list  列出当前工作目录的内容 
- mkdir <foder_name> make directory  创建文件夹
- cd  change directory 切换文件夹
- touch <file_name> 创建文件
- rm <file_path>   remove  删除文件
- rm -r <folder_path> -r  表示递归   删除文件夹
- tab 自动补全
- ctrl+shift+c/v 复制/粘贴
- !!重复上次命令
- ctrl+c 终止命令
- ctrl+z 中止命令
- sudo 管理员命令
	- apt
	- sudo apt update 只检查不更新
	- sudo apt upgrade 检查并跟新
	- sudo apt install 下载并安装远程/本地软件
		- gedit-基本文本编辑器
		- audacity-录制和编辑音频文件
		- blender-制作3D动画和可视化效果
		- gimp 编辑照片
		- nautilus-Linux文件资源管理器
		- vlc-视频播放器
	- sudo apt 删除软件并移除配置文件
		- sudo apt install ./code
