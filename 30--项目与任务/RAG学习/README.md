# RAG 学习总览

RAG（Retrieval-Augmented Generation，检索增强生成）是一种把“外部知识检索”和“大模型生成”结合起来的方法。它的核心目标不是让模型凭记忆回答，而是先从知识库中找出相关资料，再让模型基于这些资料组织答案。

## 推荐学习顺序

如果你是初学者，建议先完整读完 01 到 11，理解一个标准 RAG 系统如何工作；之后再学习 12 到 16，掌握评估、工程化、安全和高级优化；最后用 17 的路线做一个小项目。

## 一句话心法

RAG 的质量不是只由大模型决定，而是由“数据质量、切分方式、检索能力、上下文组织、生成约束、评估闭环”共同决定。
