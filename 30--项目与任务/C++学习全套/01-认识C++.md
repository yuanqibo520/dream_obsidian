# 认识 C++

## 核心概念

C++ 是一门**强调性能、资源控制和抽象能力**的系统级编程语言。它既支持接近**硬件的底层操作**，也支持**面向对象、泛型编程、函数式风格和现代工程化开发。

## 适合做什么

- 操作系统、编译器、数据库、浏览器内核。
- **游戏引擎、图形渲染、音视频处理。**
- **高性能服务端、量化交易、工业软件。**
- 嵌入式、机器人、自动驾驶等对性能和资源敏感的领域。

## 学习重点

- 理解对象、内存、生命周期。
- 熟悉标准库，不重复造基础轮子。
- 掌握 RAII、智能指针、移动语义等现代 C++ 思想。
- 建立工程意识：编译、链接、测试、调试、构建系统。

## 最小示例

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, C++!\n";
    return 0;
}
```

## 常见误区

- 只学语法，不理解**内存和生命周期。**
- 过早沉迷复杂模板。
- 把 C++ 当成“带类的 C”。
- 不用标准库，手写大量低质量基础设施。
