# 智能指针

## unique_ptr

独占所有权。

```cpp
auto p = std::make_unique<User>("Alice");
```

不能拷贝，可以移动。

## shared_ptr

共享所有权。

```cpp
auto p = std::make_shared<User>("Bob");
auto q = p;
```

引用计数为 0 时对象销毁。

## weak_ptr

弱引用，不增加引用计数，常用于打破 `shared_ptr` 循环引用。

## 选择建议

- 默认使用 `std::unique_ptr`。
- 确实需要共享生命周期时使用 `std::shared_ptr`。
- 不拥有对象时使用裸指针、引用或 `std::weak_ptr`，根据语义选择。
