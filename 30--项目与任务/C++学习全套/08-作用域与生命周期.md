# 作用域与生命周期

## 作用域

作用域决定名字在哪里可见。

```cpp
int global_value = 1;

void f() {
    int local_value = 2;
}
```

`global_value` 是全局作用域，`local_value` 是局部作用域。

## 生命周期

生命周期决定对象从何时创建到何时销毁。

```cpp
void f() {
    std::string name = "cpp";
}
```

`name` 在进入函数时创建，离开函数时销毁。

## static 局部变量

```cpp
int next_id() {
    static int id = 0;
    return ++id;
}
```

`static` 局部变量只初始化一次，生命周期持续到程序结束。

## 建议

尽量缩小变量作用域，让变量靠近使用位置。
