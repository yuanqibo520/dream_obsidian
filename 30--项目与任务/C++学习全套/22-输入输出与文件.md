# 输入输出与文件

## 标准输出

```cpp
std::cout << "hello" << '\n';
```

## 标准输入

```cpp
int age;
std::cin >> age;
```

## 读取整行

```cpp
std::string line;
std::getline(std::cin, line);
```

## 文件读取

```cpp
#include <fstream>

std::ifstream file("data.txt");
std::string line;

while (std::getline(file, line)) {
    std::cout << line << '\n';
}
```

## 文件写入

```cpp
std::ofstream out("result.txt");
out << "hello\n";
```

## 建议

打开文件后检查是否成功。

```cpp
if (!file) {
    throw std::runtime_error("failed to open file");
}
```
