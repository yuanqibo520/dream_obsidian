# 模板

## 函数模板

```cpp
template <typename T>
T max_value(T a, T b) {
    return a > b ? a : b;
}
```

模板让函数可以适配不同类型。

## 类模板

```cpp
template <typename T>
class Box {
public:
    explicit Box(T value) : value_(value) {}
    T value() const { return value_; }

private:
    T value_;
};
```

## 模板实例化

```cpp
auto a = max_value(1, 2);
auto b = max_value(1.5, 2.5);
```

编译器会根据使用情况生成对应版本。
