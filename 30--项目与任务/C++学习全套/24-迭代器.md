# 迭代器

## 核心概念

迭代器像指针一样指向容器中的元素，是容器和算法之间的桥梁。

```cpp
std::vector<int> nums = {1, 2, 3};

auto it = nums.begin();
std::cout << *it << '\n';
```

## begin 和 end

- `begin()` 指向第一个元素。
- `end()` 指向最后一个元素之后的位置。

```cpp
for (auto it = nums.begin(); it != nums.end(); ++it) {
    std::cout << *it << '\n';
}
```

## 迭代器失效

容器插入、删除、扩容可能导致已有迭代器失效。

## 建议

能用 range-for 时优先用 range-for，需要配合算法时再显式使用迭代器。
