# 指针

## 核心概念

指针保存对象的地址。

```cpp
int x = 10;
int* p = &x;
```

`&x` 取得地址，`p` 保存地址。

## 解引用

```cpp
*p = 20;
```

`*p` 表示访问指针指向的对象。

## 空指针

```cpp
int* p = nullptr;
```

现代 C++ 使用 `nullptr`，不要使用 `NULL`。

## 指针风险

- 悬空指针：指向已经销毁的对象。
- 空指针解引用：程序崩溃或未定义行为。
- 内存泄漏：申请的资源没有释放。

## 建议

业务代码中优先使用引用、容器和智能指针，少直接管理裸指针。
