# 运算符重载

## 目的

运算符重载让自定义类型可以像内置类型一样自然使用。

```cpp
struct Vec2 {
    double x;
    double y;
};

Vec2 operator+(const Vec2& a, const Vec2& b) {
    return {a.x + b.x, a.y + b.y};
}
```

## 输出运算符

```cpp
std::ostream& operator<<(std::ostream& os, const Vec2& v) {
    return os << '(' << v.x << ", " << v.y << ')';
}
```

## 建议

- 运算符语义要符合直觉。
- 不要滥用重载制造奇怪语法。
- 对称二元运算符常写成非成员函数。
