# 异常处理

## 基本形式

```cpp
try {
    risky();
} catch (const std::exception& e) {
    std::cerr << e.what() << '\n';
}
```

## 抛出异常

```cpp
if (value < 0) {
    throw std::invalid_argument("value must be non-negative");
}
```

## 异常安全

异常安全关注异常发生时程序是否仍保持有效状态。

常见等级：

- 基本保证：对象仍可用，没有资源泄漏。
- 强保证：操作失败时状态回滚。
- 不抛保证：函数承诺不抛异常。

## noexcept

```cpp
void close() noexcept;
```

`noexcept` 表示函数不应抛出异常，常用于移动操作和析构函数。
