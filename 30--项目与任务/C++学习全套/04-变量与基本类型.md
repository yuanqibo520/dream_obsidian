# 变量与基本类型

## 变量

变量是有名字的存储位置。

```cpp
int age = 18;
double price = 19.9;
char grade = 'A';
bool ok = true;
```

## 常见基本类型

- `int`：整数。
- `long long`：更大范围的整数。
- `float`：单精度浮点数。
- `double`：双精度浮点数。
- `char`：字符。
- `bool`：布尔值。

## const

`const` 表示不可修改。

```cpp
const double pi = 3.14159;
```

## auto

`auto` 让编译器推导类型。

```cpp
auto count = 10;      // int
auto name = "cpp";    // const char*
```

## 类型转换

```cpp
double x = 3.8;
int y = static_cast<int>(x);
```

优先使用 `static_cast`，少用 C 风格强制转换。
