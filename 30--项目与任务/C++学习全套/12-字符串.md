# 字符串

## C 字符串

```cpp
const char* s = "hello";
```

C 字符串以 `\0` 结尾，操作时容易出错。

## std::string

```cpp
#include <string>

std::string name = "C++";
name += " language";
```

现代 C++ 优先使用 `std::string`。

## 常用操作

```cpp
std::string s = "hello";

s.size();
s.empty();
s.substr(1, 3);
s.find("ell");
s.append("!");
```

## 遍历

```cpp
for (char ch : s) {
    std::cout << ch << '\n';
}
```

## 注意

`std::string::size()` 返回无符号整数，和有符号整数比较时要小心警告。
