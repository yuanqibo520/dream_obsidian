# 编译、链接与库

## 编译流程

C++ 程序通常经历：

1. 预处理。
2. 编译。
3. 汇编。
4. 链接。

## 翻译单元

一个 `.cpp` 文件经过预处理后形成一个翻译单元。

## 声明与定义

声明告诉编译器名字存在，定义分配实体或提供实现。

```cpp
int add(int a, int b); // 声明

int add(int a, int b) { // 定义
    return a + b;
}
```

## 链接错误

常见链接错误：

- 函数声明了但没有定义。
- 定义重复。
- 忘记链接需要的库。

## 静态库与动态库

- 静态库在链接时合入可执行文件。
- 动态库在运行时加载。
