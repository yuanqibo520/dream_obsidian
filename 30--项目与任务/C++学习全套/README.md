# C++ 学习全套目录

这套笔记按“一个文件覆盖一个知识点”的方式组织，适合从零基础到工程实践逐步学习。

## 学习顺序

1. 先学语言基础：语法、类型、表达式、流程控制、函数。
2. 再学内存模型：数组、指针、引用、字符串、动态内存。
3. 进入面向对象：类、构造析构、封装、继承、多态。
4. 学习泛型和 STL：模板、容器、迭代器、算法、函数对象。
5. 学现代 C++：RAII、智能指针、移动语义、Lambda、并发。
6. 补工程能力：编译链接、CMake、调试、测试、代码规范。
7. 用项目巩固：命令行工具、小游戏、数据结构、网络或图形项目。
