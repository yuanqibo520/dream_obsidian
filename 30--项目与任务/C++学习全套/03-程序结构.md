# 程序结构

## 最小程序

```cpp
#include <iostream>

int main() {
    std::cout << "Hello\n";
    return 0;
}
```

## 组成部分

- `#include`：引入头文件。
- `main`：程序入口。
- `{}`：代码块。
- `return 0`：表示程序正常结束。

## 头文件与源文件

头文件通常放声明：
	**头文件**一般是 .h 或 .hpp，用来写“声明”：告诉别的文件这里有什么东西可以用，比如函数名、类名、结构体、常量等。它像目录或说明书。

```cpp
// math_utils.h
int add(int a, int b);
```

源文件通常放定义：
	**源文件**一般是 .cpp，用来写“实现”：真正写函数、类的方法到底怎么工作。

```cpp
// math_utils.cpp
#include "math_utils.h"

int add(int a, int b) {
    return a + b;
}
```

## 命名空间
**命名空间**就是给名字加一个“分组/姓氏”，避免不同地方的名字撞车。

比如两个人都写了一个函数叫 `print`：

```cpp
namespace school {
    void print() {
        // 学校的 print
    }
}

namespace company {
    void print() {
        // 公司的 print
    }
}

namespace game {
    int score = 100;
}
```

使用时要写清楚你要哪个：

```cpp
school::print();
company::print();
std::cout << game::score << std::endl;
```

这里的 `::` 叫**作用域解析运算符**，可以理解成“里面的”。

最常见的是：

```cpp
std::cout << "Hello" << std::endl;
```

意思是：使用 `std` 这个命名空间里的 `cout`。`std` 是 C++ 标准库的命名空间，像 `cout`、`cin`、`string`、`vector` 很多标准库名字都在里面。

也可以写：

```cpp
using namespace std;

cout << "Hello";
```

这样就不用每次写 `std::`。但初学时我更建议先写 `std::cout`、`std::cin`，因为更清楚，也能避免名字冲突。

一句话记：**命名空间是名字的“文件夹”，`std::cout` 就是 `std` 文件夹里的 `cout`。**
