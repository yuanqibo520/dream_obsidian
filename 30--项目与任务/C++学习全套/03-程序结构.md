# 程序结构

## 最小程序

```cpp
#include <iostream>

int main() {
    std::cout << "Hello\n";
    return 0;
}
```

## 组成部分

- `#include`：引入头文件。
- `main`：程序入口。
- `{}`：代码块。
- `return 0`：表示程序正常结束。

## 头文件与源文件

头文件通常放声明：

```cpp
// math_utils.h
int add(int a, int b);
```

源文件通常放定义：

```cpp
// math_utils.cpp
#include "math_utils.h"

int add(int a, int b) {
    return a + b;
}
```

## 命名空间

```cpp
namespace app {
int version() {
    return 1;
}
}
```

命名空间用于避免名称冲突。
