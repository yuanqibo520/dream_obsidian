# 构造函数与析构函数

## 构造函数

构造函数负责初始化对象。

```cpp
class User {
public:
    User(std::string name, int age)
        : name_(std::move(name)), age_(age) {}

private:
    std::string name_;
    int age_;
};
```

优先使用成员初始化列表。

## 析构函数

析构函数负责对象销毁时的清理。

```cpp
class File {
public:
    ~File() {
        close();
    }
};
```

## 拷贝控制

如果类管理资源，需要考虑：

- 拷贝构造函数。
- 拷贝赋值运算符。
- 移动构造函数。
- 移动赋值运算符。
- 析构函数。

## Rule of Zero

如果能用标准库类型管理资源，就不要手写析构、拷贝和移动函数。
