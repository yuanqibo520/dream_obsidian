# 类与对象

## 类

类把数据和操作数据的函数组织在一起。

```cpp
class Counter {
public:
    void increment() {
        ++value_;
    }

    int value() const {
        return value_;
    }

private:
    int value_ = 0;
};
```

## 对象

```cpp
Counter counter;
counter.increment();
std::cout << counter.value() << '\n';
```

## 访问控制

- `public`：类外可以访问。
- `private`：只有类内部可以访问。
- `protected`：类内部和派生类可以访问。

## const 成员函数

```cpp
int value() const;
```

`const` 成员函数承诺不修改对象的可观察状态。
