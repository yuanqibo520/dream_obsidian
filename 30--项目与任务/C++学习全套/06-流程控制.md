# 流程控制

## if

```cpp
if (score >= 90) {
    grade = 'A';
} else if (score >= 60) {
    grade = 'B';
} else {
    grade = 'F';
}
```

## switch

```cpp
switch (choice) {
case 1:
    start();
    break;
case 2:
    stop();
    break;
default:
    help();
    break;
}
```

## for

```cpp
for (int i = 0; i < 10; ++i) {
    std::cout << i << '\n';
}
```

## while

```cpp
while (running) {
    update();
}
```

## range-for

```cpp
for (const auto& item : items) {
    std::cout << item << '\n';
}
```

遍历容器时优先使用 range-for。
