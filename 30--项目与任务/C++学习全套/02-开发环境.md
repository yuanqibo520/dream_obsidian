# 开发环境

## 编译器

常见 C++ 编译器：

- GCC：Linux 常用，命令是 `g++`。
- Clang：错误提示友好，跨平台能力强。
- MSVC：Windows Visual Studio 默认编译器。

## 命令行编译

```bash
g++ main.cpp -std=c++20 -Wall -Wextra -O2 -o app
./app
```

参数含义：

- `-std=c++20`：使用 C++20 标准。
- `-Wall -Wextra`：打开常见警告。
- `-O2`：开启优化。
- `-o app`：指定输出文件名。

## IDE 和编辑器

- Visual Studio：适合 Windows 大型 C++ 项目。
- CLion：跨平台，CMake 支持好。
- VS Code：轻量，需要配置编译器和插件。

## 建议

初学时先会命令行编译，再使用 IDE。这样能理解编译、链接、运行的真实过程。
