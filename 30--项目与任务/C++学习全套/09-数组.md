# 数组

## C 风格数组

```cpp
int scores[3] = {90, 80, 70};
```

数组大小固定，访问时从下标 `0` 开始。

```cpp
scores[0] = 95;
```

## 越界问题

```cpp
scores[3] = 100; // 错误：越界
```

C++ 不会自动检查 C 风格数组越界，越界会导致未定义行为。

## std::array

```cpp
#include <array>

std::array<int, 3> scores = {90, 80, 70};
```

`std::array` 更安全，能与标准库算法配合。

## std::vector

如果长度会变化，使用 `std::vector`。

```cpp
std::vector<int> nums = {1, 2, 3};
nums.push_back(4);
```
