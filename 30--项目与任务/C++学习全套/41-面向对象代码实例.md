# 面向对象代码实例

本文件用于练习类、对象、封装、继承、多态和运算符重载。

## 示例 1：类与封装

```cpp
#include <iostream>
#include <string>

class Student {
public:
    Student(std::string name, int score)
        : name_(std::move(name)), score_(score) {}

    void set_score(int score) {
        if (score >= 0 && score <= 100) {
            score_ = score;
        }
    }

    char level() const {
        if (score_ >= 90) return 'A';
        if (score_ >= 80) return 'B';
        if (score_ >= 60) return 'C';
        return 'F';
    }

    void print() const {
        std::cout << name_ << ": " << score_ << ", " << level() << '\n';
    }

private:
    std::string name_;
    int score_;
};

int main() {
    Student student("Alice", 85);
    student.print();
    student.set_score(95);
    student.print();
}
```

检查点：

- 能解释为什么 `score_` 不直接设为 `public`。
- 能添加 `name()` 和 `score()` 两个只读接口。

## 示例 2：构造函数与析构函数

```cpp
#include <iostream>
#include <string>

class Trace {
public:
    explicit Trace(std::string name) : name_(std::move(name)) {
        std::cout << "construct " << name_ << '\n';
    }

    ~Trace() {
        std::cout << "destroy " << name_ << '\n';
    }

private:
    std::string name_;
};

void run() {
    Trace local("local");
}

int main() {
    Trace main_object("main");
    run();
    std::cout << "end\n";
}
```

检查点：

- 能写出程序输出顺序。
- 能说明局部对象什么时候析构。

## 示例 3：继承

```cpp
#include <iostream>
#include <string>

class Employee {
public:
    Employee(std::string name, int base_salary)
        : name_(std::move(name)), base_salary_(base_salary) {}

    void print_base() const {
        std::cout << name_ << " base salary: " << base_salary_ << '\n';
    }

protected:
    int base_salary() const {
        return base_salary_;
    }

private:
    std::string name_;
    int base_salary_;
};

class Sales : public Employee {
public:
    Sales(std::string name, int base_salary, int bonus)
        : Employee(std::move(name), base_salary), bonus_(bonus) {}

    int salary() const {
        return base_salary() + bonus_;
    }

private:
    int bonus_;
};

int main() {
    Sales sales("Bob", 5000, 1200);
    sales.print_base();
    std::cout << "total salary: " << sales.salary() << '\n';
}
```

检查点：

- 能解释 `protected` 与 `private` 的区别。
- 能新增一个 `Manager` 类并计算管理津贴。

## 示例 4：多态

```cpp
#include <iostream>
#include <memory>
#include <vector>

class Shape {
public:
    virtual ~Shape() = default;
    virtual double area() const = 0;
};

class Rectangle : public Shape {
public:
    Rectangle(double width, double height) : width_(width), height_(height) {}

    double area() const override {
        return width_ * height_;
    }

private:
    double width_;
    double height_;
};

class Circle : public Shape {
public:
    explicit Circle(double radius) : radius_(radius) {}

    double area() const override {
        return 3.14159 * radius_ * radius_;
    }

private:
    double radius_;
};

int main() {
    std::vector<std::unique_ptr<Shape>> shapes;
    shapes.push_back(std::make_unique<Rectangle>(3.0, 4.0));
    shapes.push_back(std::make_unique<Circle>(2.0));

    for (const auto& shape : shapes) {
        std::cout << shape->area() << '\n';
    }
}
```

检查点：

- 能解释为什么基类析构函数要是 `virtual`。
- 能新增 `Triangle` 类并加入容器。

## 示例 5：运算符重载

```cpp
#include <iostream>

struct Vec2 {
    double x = 0.0;
    double y = 0.0;
};

Vec2 operator+(const Vec2& a, const Vec2& b) {
    return {a.x + b.x, a.y + b.y};
}

Vec2 operator*(const Vec2& v, double scale) {
    return {v.x * scale, v.y * scale};
}

std::ostream& operator<<(std::ostream& os, const Vec2& v) {
    return os << '(' << v.x << ", " << v.y << ')';
}

int main() {
    Vec2 a{1.0, 2.0};
    Vec2 b{3.0, 4.0};

    std::cout << a + b << '\n';
    std::cout << a * 3.0 << '\n';
}
```

检查点：

- 能解释为什么 `operator<<` 返回 `std::ostream&`。
- 能添加 `operator-`。
