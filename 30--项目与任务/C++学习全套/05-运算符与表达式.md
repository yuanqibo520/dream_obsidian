# 运算符与表达式

## 算术运算符

```cpp
int a = 10;
int b = 3;

int sum = a + b;
int diff = a - b;
int product = a * b;
int quotient = a / b;
int remainder = a % b;
```

## 比较运算符

```cpp
a == b;
a != b;
a > b;
a <= b;
```

**比较表达式的结果是 `bool`。**

## 逻辑运算符

```cpp
bool ok = age >= 18 && has_id;
bool allowed = is_admin || is_owner;
bool denied = !allowed;
```

`&&` 和 `||` 具有短路求值特性。
```C++
##关系运算符不能连写
// 错误理解
if (0 < x < 10) { }

// 正确写法
if (0 < x && x < 10) { }
```


## 赋值运算符

```cpp
x += 1;
x -= 2;
x *= 3;
```


## 三目运算符

```C++
int maxValue = a > b ? a : b;
```

## 优先级建议

不要依赖复杂优先级。表达式一复杂，就加括号。
