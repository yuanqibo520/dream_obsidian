# 并发编程

## thread

```cpp
std::thread worker([] {
    do_work();
});

worker.join();
```

`join` 等待线程结束。

## mutex

```cpp
std::mutex m;

{
    std::lock_guard<std::mutex> lock(m);
    shared_data++;
}
```

互斥锁用于保护共享数据。

## future

```cpp
auto f = std::async(std::launch::async, [] {
    return compute();
});

int result = f.get();
```

## atomic

```cpp
std::atomic<int> counter = 0;
counter++;
```

`atomic` 适合简单共享计数或状态。

## 建议

并发代码优先减少共享状态。能用任务模型时，不要过早手写复杂锁。
