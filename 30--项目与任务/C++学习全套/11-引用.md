# 引用

## 左值引用

引用是对象的别名。

```cpp
int x = 10;
int& r = x;
r = 20;
```

修改 `r` 等同于修改 `x`。

## const 引用

```cpp
void print(const std::string& text) {
    std::cout << text << '\n';
}
```

`const T&` 避免拷贝，同时保证函数不修改对象。

## 引用与指针区别

- 引用必须初始化。
- 引用通常不能重新绑定到另一个对象。
- 指针可以为空，引用一般不表示空。

## 典型用途

- 函数参数避免拷贝。
- 函数返回已有对象。
- 实现运算符重载。
