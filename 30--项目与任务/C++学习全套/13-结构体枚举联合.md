# 结构体、枚举、联合

## struct

```cpp
struct Point {
    double x;
    double y;
};
```

`struct` 适合表达简单数据聚合。

## enum class

```cpp
enum class Color {
    Red,
    Green,
    Blue
};
```

优先使用 `enum class`，它比传统 `enum` 更安全，名称不会泄漏到外层作用域。

## union

```cpp
union Value {
    int i;
    double d;
};
```

`union` 的多个成员共用同一块内存。普通业务代码很少直接使用，现代 C++ 更常用 `std::variant`。

## 建议

- 简单数据用 `struct`。
- 有限状态用 `enum class`。
- 多类型值优先考虑 `std::variant`。
