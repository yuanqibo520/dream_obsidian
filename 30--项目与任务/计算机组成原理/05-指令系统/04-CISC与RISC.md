# CISC 与 RISC

## CISC

CISC 是复杂指令集计算机。

特点：

- 指令数量多。
- 指令长度可变。
- 寻址方式多。
- 一条指令可能完成复杂操作。
- 控制器常较复杂。

## RISC

RISC 是精简指令集计算机。

特点：

- 指令数量少。
- 指令长度固定。
- 寻址方式少。
- 大多数指令在一个周期或少数周期完成。
- 采用 Load/Store 结构。
- 适合流水线。

## Load/Store 结构

只有 LOAD 和 STORE 指令访问内存，运算指令只在寄存器之间进行。

## 对比

| 项目 | CISC | RISC |
|---|---|---|
| 指令复杂度 | 高 | 低 |
| 指令长度 | 多为可变 | 多为固定 |
| 寻址方式 | 多 | 少 |
| 流水线实现 | 较难 | 较易 |
| 编译器压力 | 较小 | 较大 |

## 常考点

RISC 不等于功能弱，而是把复杂性更多交给编译器和简单高效的硬件流水线。
