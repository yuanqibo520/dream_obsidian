# 指令格式

## 指令组成

一条机器指令通常由操作码和地址码组成。

```text
指令 = 操作码 + 地址码
```

## 操作码

操作码指出指令要完成的操作，例如加法、访存、转移。

## 地址码

地址码指出操作数地址、结果地址或下一条指令地址。

## 地址码个数

### 三地址指令

```text
OP A1, A2, A3
```

含义通常为 `A1 <- A2 OP A3`。

### 二地址指令

```text
OP A1, A2
```

含义通常为 `A1 <- A1 OP A2`。

### 一地址指令

常使用累加器作为隐含操作数。

### 零地址指令

常见于栈式机器。

## 常考点

- 指令长度影响取指和存储空间。
- 操作码扩展可以在固定指令长度中支持更多指令类型。
