# 字符与字符串编码

## ASCII

ASCII 用 7 位二进制表示 128 个字符，扩展 ASCII 常用 8 位。

常见编码：

- 字符 `0` 的 ASCII 码是 48。
- 字符 `A` 的 ASCII 码是 65。
- 字符 `a` 的 ASCII 码是 97。

## Unicode

Unicode 为全球文字分配统一码点，解决多语言字符编码问题。

## UTF-8

UTF-8 是 Unicode 的一种变长编码方式。

特点：

- 英文字符通常占 1 字节。
- 中文字符通常占 3 字节。
- 兼容 ASCII。

## 字符串存储

常见字符串表示方法：

- 以结束符标记，如 C 语言中的 `\0`。
- 显式保存长度，如许多高级语言的字符串对象。

## 易错点

- 字符编码保存的是字符的编码值，不是字符图形本身。
- 字符 `1` 和数字 1 在机器中编码不同。
