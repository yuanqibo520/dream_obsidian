# 易混概念辨析

## 机器字长、存储字长、指令字长

- 机器字长：CPU 一次处理数据的位数。
- 存储字长：一个存储单元保存的二进制位数。
- 指令字长：一条指令的二进制位数。

## MAR 与 MDR

- MAR 保存访存地址。
- MDR 保存从存储器读出或准备写入存储器的数据。

## Cache 与虚拟存储器

- Cache 解决 CPU 和主存速度差异。
- 虚拟存储器解决主存容量和程序地址空间问题。

## 有符号溢出与无符号进位

- 有符号数关注 OF。
- 无符号数关注 CF。

## 中断与 DMA

- 中断 I/O 每次传输通常仍由 CPU 执行服务程序。
- DMA 传输数据块时由 DMA 控制器直接控制主存和外设交换。

## 硬布线控制与微程序控制

- 硬布线控制速度快，修改难。
- 微程序控制灵活，速度相对慢。

## 响应时间与吞吐量

- 响应时间关注单个任务多快完成。
- 吞吐量关注单位时间完成多少任务。
