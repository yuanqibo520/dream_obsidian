# DMA 方式

## 定义

DMA 是直接存储器访问。外设和主存之间的数据块传输不需要 CPU 逐字参与。

## DMA 控制器

DMA 控制器负责：

- 接管总线。
- 给出内存地址。
- 控制读写。
- 统计传输长度。
- 传输结束后通知 CPU。

## 工作过程

1. CPU 初始化 DMA 参数。
2. 外设发出 DMA 请求。
3. DMA 控制器申请总线。
4. CPU 让出总线。
5. DMA 控制器完成数据传输。
6. DMA 结束后中断 CPU。

## 传输方式

- 停止 CPU 访问主存。
- 周期挪用。
- DMA 与 CPU 交替访存。

## 优点

- 适合高速、大批量数据传输。
- 减少 CPU 开销。

## 常考点

DMA 不是完全不需要 CPU，初始化和结束处理仍需要 CPU。
