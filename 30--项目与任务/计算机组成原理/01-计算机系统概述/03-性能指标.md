# 性能指标

## 基本指标

- 机器字长：CPU 一次能处理的二进制位数。
- 主频：CPU 时钟频率。
- CPI：每条指令平均需要的时钟周期数。
- MIPS：每秒执行百万条指令数。
- FLOPS：每秒浮点运算次数。
- 吞吐量：单位时间完成的任务数。
- 响应时间：从提交任务到得到结果的时间。

## CPU 执行时间

```text
CPU 执行时间 = 指令条数 × CPI × 时钟周期时间
CPU 执行时间 = 指令条数 × CPI / 主频
```

## 性能优化方向

- 减少指令条数：优化算法、编译优化、使用更强指令。
- 降低 CPI：流水线、缓存、乱序执行。
- 提高主频：工艺、微结构改进。

## 常见误区

- 主频高不一定性能强。
- MIPS 不能直接比较不同指令系统的机器。
- 响应时间和吞吐量不是同一个指标。

## 常考公式

```text
加速比 = 优化前执行时间 / 优化后执行时间
```
