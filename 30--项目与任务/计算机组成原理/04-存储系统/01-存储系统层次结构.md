# 存储系统层次结构

## 层次结构

从快到慢、从小到大通常为：

```text
寄存器 -> Cache -> 主存 -> 外存
```

## 设计目标

用较低成本获得接近高速存储器的访问速度和接近大容量存储器的容量。

## 局部性原理

### 时间局部性

最近访问过的数据或指令，短时间内可能再次访问。

### 空间局部性

访问某地址后，附近地址也可能很快被访问。

## 命中率

```text
命中率 = 命中次数 / 总访问次数
```

## 平均访问时间

```text
平均访问时间 = 命中时间 + 失效率 × 失效代价
```

## 常考点

Cache 和虚拟存储器都依赖局部性原理，但前者主要解决速度差异，后者主要扩展地址空间。
