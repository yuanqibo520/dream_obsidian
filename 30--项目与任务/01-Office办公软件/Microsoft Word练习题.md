## 📄 练习1：个人年度复盘报告  

**🎯 目标**：制作一份结构清晰、数据可视、可导出分享的年度复盘文档（PDF）  

### 🔧 操作步骤

#### 第一阶段：内容与结构搭建
1. **创建多级标题**  
   - 应用“标题1”：如“2025 年度复盘”  
   - 应用“标题2”：如“学习成长”“健康管理”“财务总结”“人际关系”  
   - 应用“标题3”：如“目标 vs 成果”“关键事件”“改进计划”  

2. **插入执行摘要（自动提取）**  
   - 在封面后新建一页  
   - 输入：`本年度核心成果：`  
   - 光标后插入域：`引用 → 交叉引用 → 引用类型=标题 → 选“学习成长” → 插入`  
   - 重复为其他章节插入 → 全选 → `F9` 更新  

#### 第二阶段：数据整合与可视化
3. **嵌入 Excel 动态图表**  
   - 在 Excel 中制作“月度学习时长”柱状图 → 复制  
   - Word 中：`开始 → 选择性粘贴 → Microsoft Excel 图表对象`  
   - 双击图表可回 Excel 编辑（保持链接）  

4. **美化视觉元素**  
   - 插入 → 形状 → 圆角矩形作为“关键成果”卡片  
   - 插入 → 图标（Word 365 内置）添加 📚 💪 💰 等符号  
   - 用“文本框 + 浅灰底纹”突出反思语句  

#### 第三阶段：自动化与输出
5. **生成目录**  
   - 光标定位 → `引用 → 目录 → 自动目录`  
   - 修改样式：目录 1/2/3 对应标题 1/2/3  

6. **导出 PDF**  
   - `文件 → 导出 → 创建 PDF`  
   - 勾选“优化最小文件大小”（适合分享）  

### ✅ 核心要求（验收清单）
- [ ] 所有章节标题用**样式**（非手动加粗）  
- [ ] 执行摘要内容**自动从正文章节提取**（非手打）  
- [ ] 至少 1 个 **可编辑的 Excel 图表**（双击能改数据）  
- [ ] 目录点击可**跳转**，PDF 中保留超链接  
- [ ] 全文无手动空格/回车调格式（用段落缩进/间距控制）

## 📄 练习2：远程协作项目说明书  
**🎯 目标**：产出一份支持多人修订、结构清晰、跨设备可读的协作文档  

### 🔧 操作步骤

#### 第一阶段：文档结构与分节
1. **插入分节符**  
   - 封面页末尾 → `布局 → 分隔符 → 下一页`  
   - 目录页末尾 → 再插“下一页”分节符  
   - 正文开头 → 新节  

2. **设置页码格式**  
   - 封面：**无页眉页脚**  
   - 目录节：页脚 → 页码 → “i, ii, iii...”  
   - 正文节：页脚 → 页码 → “1, 2, 3...”，首页不同（第一章无页码）  

3. **断开节链接**  
   - 进入目录节页脚 → 取消 **“链接到前一节”**  
   - 进入正文节页脚 → 同样取消链接  

#### 第二阶段：协作功能配置
4. **开启修订模式**  
   - `审阅 → 修订 → 所有标记`  
   - 模拟 3 人修改：  
     - 用不同账户（或改用户名：`文件 → 选项 → 常规 → 用户名`）  
     - 各自插入批注（`审阅 → 新建批注`）  

5. **添加导航元素**  
   - 目录后插入 **“返回顶部”按钮**：  
     - 插入 → 书签 → 命名 `Top`（放在封面标题处）  
     - 在每章末尾：插入 → 超链接 → 本文档位置 → 选 `Top`  

#### 第三阶段：输出与兼容性
6. **导出双版本**  
   - **清洁版**：`审阅 → 接受 → 接受所有修订` → 保存为 `项目说明书_终版.docx`  
   - **留痕版**：保留修订 → 保存为 `项目说明书_修订版.docx`  
   - **Web版**：`文件 → 另存为 → 网页 (*.mht)`  

### ✅ 核心要求（验收清单）
- [ ] 封面无页码，目录用罗马数字，正文用阿拉伯数字  
- [ ] 三处“链接到前一节”**均已断开**（否则页码混乱）  
- [ ] 批注包含**具体建议**（如“此处需补充数据来源”）  
- [ ] “返回顶部”超链接**实际可点击跳转**  
- [ ] .mht 文件在手机浏览器中**可正常显示**

## 📄 练习4：技术岗求职材料包  
**🎯 目标**：用主控文档整合简历+作品说明+项目复盘，支持一键切换岗位方向  

### 🔧 操作步骤

#### 第一阶段：主控文档搭建
1. **创建主文档**  
   - 新建 Word → `视图 → 大纲 → 显示文档 → 插入子文档`  
   - 创建 3 个一级标题：  
     - `简历`  
     - `作品说明`  
     - `项目复盘`  

2. **拆分子文档**  
   - 选中“简历”标题 → `大纲 → 插入子文档` → 保存为 `简历.docx`  
   - 重复为另两部分创建子文档（保存在同一文件夹）  

#### 第二阶段：动态简历（VBA 宏）
3. **设计简历模板**  
   - 在“简历.docx”中：  
     - 姓名：大号黑体  
     - 电话：灰色（RGB 128,128,128）  
     - 技能标签：用制表位对齐（非空格）  

4. **添加岗位切换宏**  
   - `开发工具 → Visual Basic`  
   - 在 `ThisDocument` 中粘贴以下代码（简化版）：
     ```vba
     Sub SwitchToBackend()
         With ActiveDocument.Content.Find
             .Text = "前端"
             .Replacement.Text = "后端"
             .Execute Replace:=wdReplaceAll
         End With
         MsgBox "已切换为后端简历"
     End Sub
     ```
   - 保存为 **启用宏的模板**（`.dotm`）

#### 第三阶段：作品与项目增强
5. **插入代码截图与题注**  
   - 截图 → 插入 → 题注 → 标签选“图” → 勾选“包含章节号”  
   - 在“项目复盘”中引用：`引用 → 交叉引用 → 图 → 选对应图`  

6. **导出安全 PDF**  
   - `文件 → 导出 → 创建 PDF`  
   - 工具 → 安全性选项 → 勾选 **“删除文档属性和个人信息”**  

### ✅ 核心要求（验收清单）
- [ ] 主文档中 **3 个子文档可独立打开编辑**  
- [ ] 简历中 **岗位关键词可通过宏一键替换**（非手改）  
- [ ] 所有图/表题注 **带章节号**（如“图 2.1”）  
- [ ] 交叉引用 **可更新**（修改图标题后 `Ctrl+A → F9` 同步）  
- [ ] PDF 导出后 **无作者/路径等元数据泄露**

## 🆘 遇到困难？这样快速定位
| 问题现象 | 可能原因 | 快速检查点 |
|--------|--------|----------|
| 目录/题注编号错乱 | 未用样式/多级列表 | 检查标题是否应用“标题1”等内置样式 |
| 页码不按节变化 | 未断开“链接到前一节” | 进入页脚，看“链接到前一节”是否高亮 |
| 宏按钮点不动 | 宏被禁用 | 文件 → 选项 → 信任中心 → 启用宏（仅限自己文件） |
| 图表无法编辑 | 粘贴为图片 | 用“选择性粘贴 → Excel 图表对象” |
| 子文档无法更新 | 主文档未保存 | 先保存主文档，再修改子文档 |
