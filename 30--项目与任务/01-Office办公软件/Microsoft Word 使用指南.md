## 二、操作要点

### 功能区与视图设置
- **开始选项卡** → 勾选 **“显示编辑标记”**（¶）：查看段落、空格、换行符。
- **视图** → 勾选 **“标尺”**：辅助排版。
  - **标尺三角形/矩形含义**：
    - 上三角：首行缩进
    - 下三角：悬挂缩进
    ⬇ 矩形：左缩进
    ➡ 右侧三角：右缩进

### 段落与格式
- **段前/段后间距**：受“对齐到网格”影响，若过大可 **关闭“对齐到网格”**
- **避免孤行**：最后一段符号单独成页？→ 调整 **页面下边距** 或 使用“段中不分页”。
- **换行 vs 段落**：
  - `Enter`：插入段落标记（应用段落格式）
  - `Shift + Enter`：插入换行符（同段内换行）

## 四、邮件合并（批量生成文档）
	1. **邮件** 选项卡 → **选择收件人** → 使用现有 Excel 列表（**必须含表头**）
1. 插入 **合并域**（如 «姓名»）替换占位符，即可一键对多文本排版
2. **完成并合并** → 生成新文档（或直接发送邮件）
   - 邮件格式选 **HTML** 可保留格式（需 Outlook 默认邮箱）

## 五、格式刷
- **单次使用**：点击格式刷 → 应用到目标段落
- **多次使用**：**双击格式刷** → 连续应用 → 再次点击或按 `Esc` 退出
- **技巧**：将光标置于源段落内（勿选中段落标记 ¶），可精准复制段落格式

## 六、修订与批注（审阅协作）
- **修订**：开启后，所有修改以标记形式显示，可 **接受/拒绝**
- **批注**：右侧“小纸条”，可删除或回复
- **比较文档**：审阅 → 比较 → 识别两版差异

## 七、简历排版技巧
- 姓名：大写 + 大号字体
- 电话：浅灰色（`#808080`）
- 项目符号与文字间距过大？→ 调整 **悬挂缩进**
- **对齐制表位**：
  - 插入制表符（`Tab`）→ 段落中对齐 → 可用格式刷复制
  - 优势：比空格更精准、专业

## 八、字体建议
| 语言 | 推荐字体 |
|------|--------|
| 中文 | 华文细黑、微软雅黑、宋体、黑体 |
| 英文 | Times New Roman、Calibri、Arial |

- `Ctrl + D`：打开字体对话框，对**中英文字体可分别设置**
- 大标题建议用 **黑体**

## 九、高级查找与替换（通配符）
- 打开 `Ctrl + H` → 勾选 **“使用通配符”**
- 常用符号：
  - `[0-9]`：0到9的任意数字
  - `?`：任意单个字符
  - `*`：任意多个字符
  - `( )`：分组表达式
  - `{n}`：重复 n 次
  - `\1, \2`：引用第1/2个分组
- 可查找 **带格式文本**（如红色、带下划线）、**图片**（嵌入式）

## 十、表格操作
- **选中表格**：不影响其他表格
- **文本转表格**：使用中文分隔符（如顿号、逗号）需注意一致性（在对话框中为英文输入法）
- ==**移动行**==：`Alt + Shift + ↑/↓`
- **排序**：支持按金额、日期排序（需更新域）
- **跨页表格**：
  - 布局 → **重复标题行**
  - 允许 **跨页断行**
- **固定列宽**：避免图片撑大表格 → 表格属性 → 列 → 指定宽度

---

## 十一、多级列表与标题
- **标准编号**：一、(一) 1. (1) 或 1 → 1.1 → 1.1.1
- **操作**：
  1. 定义多级列表（开始 → 多级列表 → 定义新）
  2. 将各级链接到 **标题1~3 样式**
  3. 编号后加 **空格**（非制表符）
- **清除多余缩进**：避免手动空格，用样式控制

---

## 十二、主控文档与子文档（长文档管理）
- **视图 → 大纲 → 显示文档 → 插入子文档**
- **协作建议**：小组讨论后制定标准样式文档 → 保存为模板（`.dotx`）统一样式
- 子文档基于 **一级标题** 拆分
- 默认为 **链接状态**，修改子文档会同步源文件
- **最终定稿**：取消链接 → 转为普通文档
- 合并用 **插入 → 对象 → 文件中的文字**
- 对规范段落应用 **“标题样式”**（如标题1、正文），后续可一键复用。
- 使用 **F12另存为 `.dotx` 模板文件**，供团队统一格式。
- **密码保护**：另存为时 → 工具 → 常规选项 → 设置打开/修改密码。

---

## 十三、目录与页眉页脚
- **目录**：引用 → 目录 → 自动基于标题样式生成
- **页眉页脚分节**：
  - 目录与正文需不同页码？→ 插入 **分节符**
  - 取消 **“链接到前一节”** 再分别设置
  - 设置奇偶页不同
- **自动标题/页码**：
  - 页眉：插入 **“StyleRef” 域** 引用二级标题
  - 页脚：`Page`（当前页） + `NumPages`（总页数）
- **封面**：首页不同 → 勾选 **“首页不同”**

---

## 十四、封面设计
- 基本信息：使用 **分散对齐** + 下划线（或表格线）
- 固定图片位置：**插入无边框表格**，将图片置于单元格中
- 通过形状填充颜色，合并图形来设计风格

## 十五、题注与交叉引用
- 原文内（交叉引用）
- 交叉引用 → 引用类型选 **“图”/“表”** → 插入编号
- 更新：全选文档 → `F9`
- **题注**（引用 → 插入题注）：
  - 勾选 **“包含章节号”**（需多级列表）
  - 修改后按 `Ctrl + A` → `F9` **更新域**

## 十六、宏（自动化）
- 启用开发工具：文件 → 选项 → 自定义功能区 → 勾选“开发工具”
- 宏安全：**信任中心 → 宏设置 → 禁用所有宏并警告**
- 录制宏 → 保存到 `Normal.dotm`（全局模板）
- **风险**：`.docm` / `.dotm` 可能含宏病毒，勿随意启用
- **AI辅助**：可生成 VBA 代码，但需人工审核

## 十七、大模型处理 Word 的局限性
> ✅ 可行：生成内容草稿、提供排版步骤建议  
> ❌ 不可行：直接操作 Word 格式 / 样式 / 域

### 核心痛点：
1. **无法直接读写 `.docx`**（本质是 ZIP 包，含 XML，模型无法解析）
2. **格式语义不可见**：模型看不到“标题1”“段前12磅”等样式信息
3. **用户指令模糊**：如“排版好看”缺乏可执行标准
4. **动态交互缺失**：Word 依赖“域”“样式”“节”等动态机制，模型无法触发
5. **典型失败场景**：
   - “插入图后自动更新目录” → 需手动 `F9` 更新域
   - “自动重编号后续图” → 需 Word 内部交叉引用机制
