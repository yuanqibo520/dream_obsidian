# 递归基础

递归是一种把问题分解为同类子问题的方法。

## 递归三要素

1. 递归定义：函数表示什么含义。
2. 递归终止条件：什么时候停止。
3. 递归转移：如何把当前问题变成更小问题。

## 示例：阶乘

```text
factorial(n) = n * factorial(n - 1)
factorial(0) = 1
```

## 示例：树的遍历

树天然适合递归，因为每棵子树结构相同。

```text
遍历当前节点
递归遍历左子树
递归遍历右子树
```

## 易错点

- 没有终止条件导致无限递归。
- 递归参数没有向终止条件靠近。
- 只关注代码执行顺序，忽略函数语义。

## 递归与回溯

递归负责进入下一层问题，回溯负责撤销当前选择并尝试其他选择。
