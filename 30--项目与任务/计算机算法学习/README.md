# 计算机算法学习

这是一个按模块拆分的算法学习知识库。每个 Markdown 文件覆盖一个相对独立的知识点，适合配合 Obsidian、VS Code 或任意 Markdown 阅读器使用。

## 推荐学习顺序

1. 先学复杂度分析、递归、基础数据结构。
2. 再学排序、二分、双指针、滑动窗口。
3. 进入树、堆、哈希、并查集、图论。
4. 重点攻克动态规划、贪心、回溯。
5. 最后学习字符串算法、数学算法、进阶专题。

## 使用方法

- 每读完一个知识点，自己补充 1 到 3 道典型题。
- 每个算法都建议写出：适用场景、核心不变量、复杂度、易错点。
- 不要只背模板，要能解释为什么算法正确。
