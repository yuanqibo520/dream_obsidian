# 深度优先搜索 DFS

DFS 沿着一条路径尽可能深入，无法继续时回退。

## 常见应用

- 连通块计数
- 岛屿问题
- 拓扑排序
- 图中环检测
- 回溯搜索

## 基本结构

```text
dfs(node):
    标记 node 已访问
    for next in neighbors[node]:
        if next 未访问:
            dfs(next)
```

## 状态标记

有向图环检测常用三色标记：

- 0：未访问
- 1：访问中
- 2：已完成

## 易错点

- 忘记 visited。
- 递归深度过大导致栈溢出。
- 有向图和无向图环判断方式不同。
