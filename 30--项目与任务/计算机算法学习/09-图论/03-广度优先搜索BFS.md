# 广度优先搜索 BFS

BFS 按层扩展，通常使用队列。

## 常见应用

- 最短步数问题
- 层序遍历
- 网格扩散
- 多源 BFS
- 无权图最短路

## 基本结构

```text
queue 放入起点
标记起点已访问
while queue 非空:
    node = queue 出队
    for next in neighbors[node]:
        if next 未访问:
            标记 next
            next 入队
```

## 多源 BFS

把多个起点同时入队，然后一起扩散。常用于腐烂橘子、地图距离等问题。

## 易错点

- 应在入队时标记访问，而不是出队时。
- 层数统计时要固定当前层队列长度。
- 网格边界判断遗漏。
