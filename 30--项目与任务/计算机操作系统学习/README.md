# 计算机操作系统学习

这是一套按模块拆分的操作系统学习笔记。每个 Markdown 文件覆盖一个核心知识点，适合课堂学习、考研复习、面试准备和系统性查漏补缺。

## 学习路径

1. 先读 `01-操作系统导论`，建立整体框架。
2. 再读 `02-进程与线程`、`03-CPU调度`、`04-同步与互斥`，掌握并发的核心。
3. 继续学习 `05-内存管理` 和 `06-文件系统`，理解资源抽象与管理。
4. 最后学习 `07-输入输出与设备`、`08-死锁安全与虚拟化`，补齐系统实现和工程视角。
