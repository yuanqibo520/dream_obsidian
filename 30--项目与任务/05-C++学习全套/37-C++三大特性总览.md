# C++ 三大特性总览

## 三大特性是什么

C++ 面向对象的三大核心特性通常指：

- 封装：把数据和操作数据的函数组合在一起，并隐藏内部实现。
- 继承：让一个类基于另一个类扩展能力，表达 is-a 关系。
- 多态：用统一接口调用不同对象的具体行为。

## 封装

封装关注“对象应该暴露什么，不应该暴露什么”。

```cpp
class Account {
public:
    void deposit(int amount) {
        if (amount > 0) {
            balance_ += amount;
        }
    }

    int balance() const {
        return balance_;
    }

private:
    int balance_ = 0;
};
```

这里 `balance_` 被隐藏起来，外部只能通过 `deposit` 和 `balance` 访问对象状态。

## 继承

继承关注“类型之间是不是一种更具体的关系”。

```cpp
class Shape {
public:
    virtual ~Shape() = default;
    virtual double area() const = 0;
};

class Rectangle : public Shape {
public:
    Rectangle(double w, double h) : w_(w), h_(h) {}

    double area() const override {
        return w_ * h_;
    }

private:
    double w_;
    double h_;
};
```

`Rectangle` 是一种 `Shape`，所以可以使用 public 继承。

## 多态

多态关注“同一调用在不同对象上表现出不同行为”。

```cpp
void print_area(const Shape& shape) {
    std::cout << shape.area() << '\n';
}
```

`print_area` 不需要知道传入的是矩形、圆形还是其他形状，只依赖 `Shape` 的抽象接口。

## 三者关系

- 封装让类内部状态可靠。
- 继承建立抽象和具体类型之间的关系。
- 多态让程序面向抽象接口编写，减少对具体类型的依赖。

## 学习提醒

不要为了使用继承而使用继承。如果只是代码复用，优先考虑组合；如果确实存在稳定的 is-a 关系，再使用继承和多态。
