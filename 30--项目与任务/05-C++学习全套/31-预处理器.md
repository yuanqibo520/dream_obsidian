# 预处理器

## include

```cpp
#include <iostream>
#include "my_header.h"
```

尖括号通常用于标准库或系统头文件，引号通常用于项目头文件。

## define

```cpp
#define MAX_SIZE 100
```

现代 C++ 中常用 `constexpr` 替代简单宏常量。

```cpp
constexpr int max_size = 100;
```

## 条件编译

```cpp
#ifdef DEBUG
std::cout << "debug\n";
#endif
```

## include guard

```cpp
#ifndef MY_HEADER_H
#define MY_HEADER_H

// declarations

#endif
```

也可以使用 `#pragma once`。

## 建议

少用宏。宏没有类型检查，容易污染命名空间。
