# 移动语义

## 为什么需要移动

有些对象拷贝成本高，例如 `std::vector`、`std::string`。移动语义允许把资源从一个对象转移到另一个对象，避免昂贵拷贝。

## 右值引用

```cpp
void consume(std::string&& s);
```

`T&&` 可以绑定到临时对象。

## std::move

```cpp
std::string a = "hello";
std::string b = std::move(a);
```

`std::move` 本身不移动，它把对象转换成可被移动的表达式。

## 移动后对象

移动后的对象仍然有效，但值通常不应依赖。

## 完美转发

```cpp
template <typename T>
void wrapper(T&& value) {
    target(std::forward<T>(value));
}
```

完美转发用于保留参数的左值或右值属性。
