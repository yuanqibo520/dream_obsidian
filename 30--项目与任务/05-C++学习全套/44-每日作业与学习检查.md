# 每日作业与学习检查

这份计划按 30 天安排。每天都包含学习目标、编码作业和验收标准。建议每天至少写 30 到 90 分钟代码。

## 使用方法

每天完成后，在对应条目前打勾，并写一句复盘：

```text
日期：
完成情况：
今天卡住的问题：
明天要补的点：
```

## 第 1 天：环境与第一个程序

学习内容：

- 编译器、IDE、命令行编译。
- `main` 函数、`#include`、`std::cout`。

作业：

- 写一个程序输出姓名、年龄、学习目标。
- 用命令行完成一次编译和运行。

验收标准：

- 能独立编译 `.cpp` 文件。
- 能解释源文件、可执行文件的区别。

## 第 2 天：变量与基本类型

作业：

- 写一个 BMI 计算器。
- 输入身高和体重，输出 BMI 和简单判断。

验收标准：

- 能使用 `int`、`double`、`bool`。
- 能使用 `static_cast` 做类型转换。

## 第 3 天：运算符与表达式

作业：

- 写一个四则运算小程序。
- 支持 `+ - * / %`。

验收标准：

- 能处理除数为 0 的情况。
- 能说出 `%` 只能用于整数。

## 第 4 天：流程控制

作业：

- 写一个成绩评级程序。
- 输入 0 到 100，输出 A/B/C/D/F。

验收标准：

- 能用 `if else` 实现。
- 能用 `switch` 实现。

## 第 5 天：循环

作业：

- 输入 `n`，输出 `1` 到 `n` 的和。
- 输出 `n!`。
- 输出 `1` 到 `n` 的所有偶数。

验收标准：

- 能正确选择 `for` 和 `while`。
- 能处理 `n <= 0`。

## 第 6 天：函数

作业：

- 写 `is_prime` 判断质数。
- 写 `count_prime` 统计范围内质数数量。

验收标准：

- 能区分函数声明和定义。
- 能解释按值传参和按引用传参。

## 第 7 天：阶段复盘 1

作业：

- 做一个命令行计算器，至少支持循环输入、多种运算、退出命令。

验收标准：

- 程序能连续运行。
- 输入异常时不会直接崩溃。
- 能把逻辑拆成至少 3 个函数。

## 第 8 天：数组与 vector

作业：

- 输入一组成绩，输出最高分、最低分、平均分。

验收标准：

- 能使用 `std::vector`。
- 能避免空数组访问。

## 第 9 天：指针

作业：

- 写两个函数：一个用指针交换两个数，一个用引用交换两个数。

验收标准：

- 能解释地址、解引用、空指针。
- 能说明指针版本为什么要检查 `nullptr`。

## 第 10 天：引用与 const

作业：

- 写一个函数打印字符串长度，参数使用 `const std::string&`。
- 写一个函数修改整数，参数使用 `int&`。

验收标准：

- 能说明 `const T&` 的好处。
- 能区分引用和指针。

## 第 11 天：字符串

作业：

- 写一个文本统计器，统计字母、数字、空格数量。

验收标准：

- 能使用 `std::getline`。
- 能遍历 `std::string`。

## 第 12 天：结构体与枚举

作业：

- 定义 `Student` 结构体，包含姓名、分数、等级。
- 使用 `enum class` 表示等级。

验收标准：

- 能解释 `struct` 和 `enum class` 的作用。
- 能把成绩转换为枚举等级。

## 第 13 天：类与封装

作业：

- 写一个 `BankAccount` 类。
- 支持存款、取款、查询余额。

验收标准：

- 余额不能被外部直接修改。
- 非法金额不能改变对象状态。

## 第 14 天：构造与析构

作业：

- 写一个 `Trace` 类，在构造和析构时打印信息。
- 观察不同作用域下对象销毁顺序。

验收标准：

- 能说清局部对象的生命周期。
- 能解释析构函数什么时候执行。

## 第 15 天：继承

作业：

- 写 `Employee` 基类和 `Manager`、`Sales` 派生类。

验收标准：

- 能使用 public 继承。
- 能解释 is-a 关系。

## 第 16 天：多态

作业：

- 写 `Shape` 抽象类，派生 `Circle`、`Rectangle`、`Triangle`。
- 用统一容器保存并计算总面积。

验收标准：

- 使用 `virtual` 和 `override`。
- 基类有虚析构函数。

## 第 17 天：运算符重载

作业：

- 实现二维向量 `Vec2` 的 `+`、`-`、`*`、`<<`。

验收标准：

- 运算符语义自然。
- 输出格式清晰。

## 第 18 天：模板

作业：

- 写一个函数模板 `max_value`。
- 写一个类模板 `Box<T>`。

验收标准：

- 能用不同类型实例化模板。
- 能读懂基本模板错误。

## 第 19 天：异常处理

作业：

- 写安全除法函数，除数为 0 时抛异常。
- 在 `main` 中捕获并输出错误。

验收标准：

- 能使用 `try/catch/throw`。
- 能捕获 `const std::exception&`。

## 第 20 天：文件输入输出

作业：

- 从 `input.txt` 读取文本，统计行数、单词数、字符数。

验收标准：

- 文件打开失败时有错误提示。
- 能正确逐行读取。

## 第 21 天：阶段复盘 2

作业：

- 做一个学生成绩管理系统。
- 支持添加、删除、查询、排序、保存文件。

验收标准：

- 至少使用一个类。
- 至少使用一个 STL 容器。
- 至少有文件读写。

## 第 22 天：STL 容器

作业：

- 用 `vector` 保存数据。
- 用 `map` 或 `unordered_map` 统计词频。
- 用 `set` 去重。

验收标准：

- 能说明三个容器的适用场景。
- 能遍历并输出结果。

## 第 23 天：迭代器与算法

作业：

- 使用 `sort`、`find_if`、`count_if`、`transform`。

验收标准：

- 不手写所有循环。
- 能解释 `begin()` 和 `end()`。

## 第 24 天：Lambda

作业：

- 用 Lambda 对结构体数组按不同字段排序。
- 分别练习按值捕获和按引用捕获。

验收标准：

- 能解释捕获列表。
- 能避免返回悬空引用。

## 第 25 天：RAII 与智能指针

作业：

- 用 `unique_ptr` 管理对象。
- 用 `shared_ptr` 和 `weak_ptr` 写一个双向链表示例。

验收标准：

- 不手写 `delete`。
- 能解释所有权。

## 第 26 天：移动语义

作业：

- 写一个不可拷贝、可移动的类。
- 实现移动构造和移动赋值。

验收标准：

- 能使用 `std::move`。
- 能解释移动后对象的状态。

## 第 27 天：并发

作业：

- 开 4 个线程累加计数。
- 分别用 `mutex` 和 `atomic` 实现。

验收标准：

- 结果稳定正确。
- 能解释数据竞争。

## 第 28 天：编译流程与 CMake

作业：

- 建一个 CMake 项目。
- 至少包含 `main.cpp`、一个头文件、一个源文件。

验收标准：

- 能用 `cmake -S . -B build` 构建。
- 能解释编译错误和链接错误的区别。

## 第 29 天：调试与测试

作业：

- 给前面的计算器或成绩系统增加测试用例。
- 用断点调试一次 bug。

验收标准：

- 至少覆盖正常输入、边界输入、错误输入。
- 能描述 bug 的定位过程。

## 第 30 天：综合项目

作业：

- 完成一个命令行 Todo 工具。
- 支持新增、完成、删除、列表、保存、读取。

验收标准：

- 使用类封装任务。
- 使用 STL 容器保存任务。
- 使用文件持久化。
- 有清晰 README。
- 能独立解释核心代码。

## 每周自测问题

- 我能不能不看笔记写出当天核心代码？
- 我能不能解释每个变量的生命周期？
- 我有没有手写不必要的 `new/delete`？
- 我有没有把大函数拆成更小的函数？
- 我能不能说出这段代码可能失败在哪里？

## 最终验收标准

完成 30 天后，应能做到：

- 独立编写中小型命令行程序。
- 理解封装、继承、多态。
- 熟练使用 `vector`、`map`、`set` 和常用算法。
- 理解指针、引用、智能指针和所有权。
- 知道 C++ 从源码到可执行文件的流程。
- 能用 CMake 组织一个小项目。
