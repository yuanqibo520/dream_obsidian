# 类与对象

## 类

类把数据和操作数据的函数组织在一起。

```cpp
class Counter {
public:
    void increment() {
        ++value_;
    }

    int value() const {
        return value_;
    }

private:
    int value_ = 0;
};
```

## 对象

```cpp
Counter counter;
counter.increment();
std::cout << counter.value() << '\n';
```

## 访问控制

- `public`：类外可以访问。
- `private`：只有类内部可以访问。
- `protected`：类内部和派生类可以访问。

>派生类可以访问”指的是：**子类的成员函数里面可以访问父类的 protected 成员**，不是说外部通过子类对象就能访问。

## const 成员函数

```cpp
class Person {
public:
    int age;

    void show() const {
        cout << age << endl;
    }
};
```

函数后面的 const 表示：

> 这个成员函数承诺不会修改当前对象的数据成员
