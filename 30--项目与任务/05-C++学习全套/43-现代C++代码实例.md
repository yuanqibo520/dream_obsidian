# 现代 C++ 代码实例

本文件用于练习 RAII、智能指针、移动语义、异常和并发。

## 示例 1：RAII 管理计时

```cpp
#include <chrono>
#include <iostream>
#include <string>

class Timer {
public:
    explicit Timer(std::string name)
        : name_(std::move(name)), start_(std::chrono::steady_clock::now()) {}

    ~Timer() {
        auto end = std::chrono::steady_clock::now();
        auto ms = std::chrono::duration_cast<std::chrono::milliseconds>(end - start_);
        std::cout << name_ << " cost " << ms.count() << " ms\n";
    }

private:
    std::string name_;
    std::chrono::steady_clock::time_point start_;
};

int main() {
    Timer timer("work");

    long long sum = 0;
    for (int i = 0; i < 1000000; ++i) {
        sum += i;
    }

    std::cout << sum << '\n';
}
```

检查点：

- 能说明为什么析构函数适合做自动收尾。
- 能把计时单位改成微秒。

## 示例 2：unique_ptr 表达独占所有权

```cpp
#include <iostream>
#include <memory>
#include <string>

class User {
public:
    explicit User(std::string name) : name_(std::move(name)) {}

    void hello() const {
        std::cout << "hello, " << name_ << '\n';
    }

private:
    std::string name_;
};

std::unique_ptr<User> create_user() {
    return std::make_unique<User>("Alice");
}

int main() {
    auto user = create_user();
    user->hello();
}
```

检查点：

- 能解释为什么 `unique_ptr` 不能拷贝。
- 能用 `std::move` 把所有权转移给另一个变量。

## 示例 3：shared_ptr 与 weak_ptr

```cpp
#include <iostream>
#include <memory>
#include <string>

struct Node {
    explicit Node(std::string value) : value(std::move(value)) {}

    std::string value;
    std::shared_ptr<Node> next;
    std::weak_ptr<Node> prev;
};

int main() {
    auto first = std::make_shared<Node>("first");
    auto second = std::make_shared<Node>("second");

    first->next = second;
    second->prev = first;

    if (auto prev = second->prev.lock()) {
        std::cout << prev->value << '\n';
    }
}
```

检查点：

- 能解释 `weak_ptr` 为什么不增加引用计数。
- 能说明循环引用会造成什么问题。

## 示例 4：移动语义

```cpp
#include <iostream>
#include <string>
#include <vector>

class Buffer {
public:
    explicit Buffer(std::vector<int> data) : data_(std::move(data)) {}

    Buffer(Buffer&& other) noexcept : data_(std::move(other.data_)) {
        std::cout << "move construct\n";
    }

    Buffer& operator=(Buffer&& other) noexcept {
        if (this != &other) {
            data_ = std::move(other.data_);
        }
        std::cout << "move assign\n";
        return *this;
    }

    Buffer(const Buffer&) = delete;
    Buffer& operator=(const Buffer&) = delete;

    int size() const {
        return static_cast<int>(data_.size());
    }

private:
    std::vector<int> data_;
};

int main() {
    Buffer a({1, 2, 3});
    Buffer b(std::move(a));
    std::cout << b.size() << '\n';
}
```

检查点：

- 能解释 `std::move(a)` 后 `a` 仍然有效但值不应依赖。
- 能解释为什么移动构造函数常写 `noexcept`。

## 示例 5：异常处理

```cpp
#include <iostream>
#include <stdexcept>

int divide(int a, int b) {
    if (b == 0) {
        throw std::invalid_argument("division by zero");
    }
    return a / b;
}

int main() {
    try {
        std::cout << divide(10, 0) << '\n';
    } catch (const std::exception& e) {
        std::cerr << "error: " << e.what() << '\n';
    }
}
```

检查点：

- 能解释为什么用 `const std::exception&` 捕获。
- 能改成循环读取输入并处理异常。

## 示例 6：线程与互斥锁

```cpp
#include <iostream>
#include <mutex>
#include <thread>
#include <vector>

int main() {
    int counter = 0;
    std::mutex mutex;

    auto work = [&] {
        for (int i = 0; i < 10000; ++i) {
            std::lock_guard<std::mutex> lock(mutex);
            ++counter;
        }
    };

    std::vector<std::thread> threads;
    for (int i = 0; i < 4; ++i) {
        threads.emplace_back(work);
    }

    for (auto& thread : threads) {
        thread.join();
    }

    std::cout << counter << '\n';
}
```

检查点：

- 能解释为什么需要 `mutex`。
- 能把 `counter` 改成 `std::atomic<int>` 并移除锁。
