# 多态与虚函数

## 虚函数

```cpp
class Shape {
public:
    virtual ~Shape() = default;
    virtual double area() const = 0;
};

class Circle : public Shape {
public:
    double area() const override {
        return 3.14 * r_ * r_;
    }

private:
    double r_ = 1.0;
};
```

## 动态绑定

```cpp
void print_area(const Shape& shape) {
    std::cout << shape.area() << '\n';
}
```

通过基类引用或指针调用虚函数，会在运行时决定实际调用哪个派生类实现。

## override

派生类重写虚函数时使用 `override`，让编译器帮你检查签名是否正确。

## 虚析构函数

有虚函数的基类通常应该有虚析构函数。
