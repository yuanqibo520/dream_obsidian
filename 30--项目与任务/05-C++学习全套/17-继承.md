# 继承

## 基本形式

```cpp
class Animal {
public:
    void eat();
};

class Dog : public Animal {
public:
    void bark();
};
```

`Dog` 继承了 `Animal` 的公开接口。

## public 继承

public 继承表达 is-a 关系：派生类是一种基类。

## 访问控制

- 基类 `public` 成员在 public 继承后仍是 public。
- 基类 `protected` 成员派生类可访问。
- 基类 `private` 成员派生类不可直接访问。

## 使用建议

继承适合表达稳定的抽象层次。只是为了复用代码时，组合通常更好。
