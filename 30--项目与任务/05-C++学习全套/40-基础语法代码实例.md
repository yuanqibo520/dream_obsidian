# 基础语法代码实例

本文件用于把基础语法变成可运行的小程序。建议每个例子都手打一遍，再尝试修改输入和输出。

## 示例 1：变量、输入输出、类型转换

```cpp
#include <iostream>
#include <iomanip>

int main() {
    int age = 0;
    double height = 0.0;

    std::cout << "Age: ";
    std::cin >> age;

    std::cout << "Height in meters: ";
    std::cin >> height;

    int height_cm = static_cast<int>(height * 100);

    std::cout << "Age = " << age << '\n';
    std::cout << "Height = " << std::fixed << std::setprecision(2) << height << "m\n";
    std::cout << "Height = " << height_cm << "cm\n";
}
```

检查点：

- 能解释 `int`、`double`、`static_cast<int>` 的作用。
- 能修改程序，让它输出 BMI。

## 示例 2：if 与 switch

```cpp
#include <iostream>

int main() {
    int score = 0;
    std::cin >> score;

    if (score < 0 || score > 100) {
        std::cout << "invalid score\n";
        return 1;
    }

    char level = 'F';
    switch (score / 10) {
    case 10:
    case 9:
        level = 'A';
        break;
    case 8:
        level = 'B';
        break;
    case 7:
        level = 'C';
        break;
    case 6:
        level = 'D';
        break;
    default:
        level = 'F';
        break;
    }

    std::cout << "level = " << level << '\n';
}
```

检查点：

- 能解释 `return 1` 为什么表示异常结束。
- 能把 `switch` 改成纯 `if else`。

## 示例 3：for、while 与累加

```cpp
#include <iostream>

int main() {
    int n = 0;
    std::cin >> n;

    int sum = 0;
    for (int i = 1; i <= n; ++i) {
        sum += i;
    }

    int factorial = 1;
    int i = 1;
    while (i <= n) {
        factorial *= i;
        ++i;
    }

    std::cout << "sum = " << sum << '\n';
    std::cout << "factorial = " << factorial << '\n';
}
```

检查点：

- 能说明 `++i` 和 `i++` 在这里结果为什么一样。
- 能处理 `n <= 0` 的输入。

## 示例 4：函数拆分

```cpp
#include <iostream>

bool is_prime(int n) {
    if (n <= 1) {
        return false;
    }

    for (int i = 2; i * i <= n; ++i) {
        if (n % i == 0) {
            return false;
        }
    }

    return true;
}

int main() {
    int n = 0;
    std::cin >> n;

    if (is_prime(n)) {
        std::cout << n << " is prime\n";
    } else {
        std::cout << n << " is not prime\n";
    }
}
```

检查点：

- 能解释为什么循环条件是 `i * i <= n`。
- 能写一个函数统计 `1` 到 `n` 中有多少个质数。

## 示例 5：数组与 vector

```cpp
#include <iostream>
#include <vector>

int main() {
    std::vector<int> scores;

    for (int i = 0; i < 5; ++i) {
        int score = 0;
        std::cin >> score;
        scores.push_back(score);
    }

    int max_score = scores[0];
    int sum = 0;

    for (int score : scores) {
        if (score > max_score) {
            max_score = score;
        }
        sum += score;
    }

    double average = static_cast<double>(sum) / scores.size();

    std::cout << "max = " << max_score << '\n';
    std::cout << "average = " << average << '\n';
}
```

检查点：

- 能解释为什么 `scores[0]` 之前要保证容器非空。
- 能改成输入任意数量成绩，直到输入 `-1` 结束。

## 示例 6：指针与引用

```cpp
#include <iostream>

void add_by_pointer(int* value) {
    if (value != nullptr) {
        *value += 10;
    }
}

void add_by_reference(int& value) {
    value += 10;
}

int main() {
    int a = 1;
    int b = 1;

    add_by_pointer(&a);
    add_by_reference(b);

    std::cout << "a = " << a << '\n';
    std::cout << "b = " << b << '\n';
}
```

检查点：

- 能解释 `&a` 和 `*value`。
- 能说出什么时候更适合用引用，什么时候更适合用指针。

## 示例 7：字符串统计

```cpp
#include <iostream>
#include <string>

int main() {
    std::string text;
    std::getline(std::cin, text);

    int letters = 0;
    int digits = 0;
    int spaces = 0;

    for (char ch : text) {
        if (ch >= 'a' && ch <= 'z' || ch >= 'A' && ch <= 'Z') {
            ++letters;
        } else if (ch >= '0' && ch <= '9') {
            ++digits;
        } else if (ch == ' ') {
            ++spaces;
        }
    }

    std::cout << "letters = " << letters << '\n';
    std::cout << "digits = " << digits << '\n';
    std::cout << "spaces = " << spaces << '\n';
}
```

检查点：

- 能解释 `std::getline` 和 `std::cin >> text` 的区别。
- 能把大小写字母统计改成分别统计。
