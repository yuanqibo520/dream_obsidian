# 动态规划基础

动态规划用于解决具有重叠子问题和最优子结构的问题。

## 五步法

1. 定义状态：`dp[i]` 表示什么。
2. 写出转移：状态之间如何递推。
3. 设置初始值。
4. 确定遍历顺序。
5. 返回答案。

## 常见类型

- 一维 DP
- 二维 DP
- 背包 DP
- 区间 DP
- 树形 DP
- 状态压缩 DP

## 记忆化搜索

自顶向下递归，加缓存避免重复计算。

## 递推 DP

自底向上填表。

## 易错点

- 状态定义不清导致转移困难。
- 初始值设置错误。
- 遍历顺序不满足依赖关系。
