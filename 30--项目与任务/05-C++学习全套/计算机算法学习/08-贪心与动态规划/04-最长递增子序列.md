# 最长递增子序列

最长递增子序列，简称 LIS。

## 动态规划做法

状态：

```text
dp[i] 表示以 nums[i] 结尾的最长递增子序列长度
```

转移：

```text
如果 nums[j] < nums[i]
dp[i] = max(dp[i], dp[j] + 1)
```

时间复杂度：`O(n^2)`。

## 贪心加二分

维护数组 `tails`，其中 `tails[k]` 表示长度为 `k + 1` 的递增子序列中，最小的结尾值。

对每个数，在 `tails` 中二分找到第一个大于等于它的位置并替换。

时间复杂度：`O(n log n)`。

## 易错点

- 严格递增和非递减的二分边界不同。
- `tails` 不一定是真实子序列，但长度是正确答案。
