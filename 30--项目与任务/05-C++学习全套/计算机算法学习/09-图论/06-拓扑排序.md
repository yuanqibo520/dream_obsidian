# 拓扑排序

拓扑排序用于有向无环图，把节点排成满足依赖关系的顺序。

## 入度法

1. 统计每个节点的入度。
2. 把入度为 0 的节点入队。
3. 每次弹出一个节点，删除它的出边。
4. 如果相邻节点入度变为 0，则入队。

## 应用

- 课程表
- 任务依赖
- 编译顺序
- 判断有向图是否有环

## 判断环

如果最终排序结果数量小于节点数，说明存在环。

## 易错点

- 有向图建边方向写反。
- 没有处理多个入度为 0 的节点。
- 题目可能要求字典序最小，需要使用优先队列。
