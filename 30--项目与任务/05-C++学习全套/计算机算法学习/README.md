# 计算机算法学习

这是一个按模块拆分的算法学习知识库。每个 Markdown 文件覆盖一个相对独立的知识点，适合配合 Obsidian、VS Code 或任意 Markdown 阅读器使用。

## 推荐学习顺序

1. 先学复杂度分析、递归、基础数据结构。
2. 再学排序、二分、双指针、滑动窗口。
3. 进入树、堆、哈希、并查集、图论。
4. 重点攻克动态规划、贪心、回溯。
5. 最后学习字符串算法、数学算法、进阶专题。

## 使用方法

- 每读完一个知识点，自己补充 1 到 3 道典型题。
- 每个算法都建议写出：适用场景、核心不变量、复杂度、易错点。
- 不要只背模板，要能解释为什么算法正确。

## C++ 与力扣 Hot 100 刷题入口

建议从 `00-力扣Hot100刷题阶段表.md` 开始看。核心节奏是：

1. 学完 C++ 函数、数组、字符串、`vector` 后，正式开始刷 Hot 100。
2. 第一批只做数组、哈希、双指针、滑动窗口、前缀和。
3. 学完指针、引用、结构体后，再做链表和二叉树。
4. 学完 STL 的栈、队列、堆、哈希表后，再做栈、堆、BFS、图论。
5. 学完递归、回溯、动态规划后，再集中处理中难题。

每个知识点文件末尾都新增了“C++刷题衔接”，用于判断当前阶段该不该开始写题，以及该写 Hot 100 哪类题。
