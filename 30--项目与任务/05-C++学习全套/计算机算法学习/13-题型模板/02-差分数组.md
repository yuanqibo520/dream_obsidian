# 差分数组

差分数组用于高效处理多次区间增减。

## 一维差分

对区间 `[l, r]` 增加 `x`：

```text
diff[l] += x
diff[r + 1] -= x
```

最后对差分数组求前缀和得到原数组。

## 应用

- 区间加法
- 航班预订统计
- 拼车问题
- 多次范围更新后统一查询

## 二维差分

用于矩阵上的区域更新。

## 易错点

- `r + 1` 越界时不要更新。
- 差分适合批量更新后统一还原，不适合频繁在线查询。
- 初始数组不是全 0 时，需要先构造差分。
