# 前缀和

前缀和用于快速计算区间和。

## 一维前缀和

定义：

```text
prefix[i] 表示前 i 个元素之和
```

区间 `[l, r]` 的和：

```text
prefix[r + 1] - prefix[l]
```

## 二维前缀和

用于快速计算矩阵子区域和。

## 常见题型

- 子数组和为 k
- 区间求和
- 二维区域和
- 和可被 k 整除的子数组

## 易错点

- 前缀数组长度通常是 `n + 1`。
- 左闭右闭区间与前缀下标对应关系。
- 哈希表记录前缀和出现次数时，初始要放入 `0: 1`。
