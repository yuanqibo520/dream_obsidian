# 二分查找

二分查找用于在有序空间中快速定位目标。

## 基本条件

- 搜索空间有序。
- 能够根据中间值判断答案在左边还是右边。

## 左闭右闭模板

```text
left = 0, right = n - 1
while left <= right:
    mid = left + (right - left) / 2
    if nums[mid] == target:
        return mid
    elif nums[mid] < target:
        left = mid + 1
    else:
        right = mid - 1
```

## 二分答案

不仅数组可以二分，答案范围也可以二分。

适用场景：

- 最小化最大值。
- 最大化最小值。
- 判断函数具有单调性。

## 易错点

- 循环条件和边界更新不配套。
- `mid` 溢出。
- 查找左边界或右边界时更新方向错误。
