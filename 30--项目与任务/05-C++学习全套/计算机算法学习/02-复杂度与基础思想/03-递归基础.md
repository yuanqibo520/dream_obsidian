# 递归基础

递归是一种把问题分解为同类子问题的方法。

## 递归三要素

1. 递归定义：函数表示什么含义。
2. 递归终止条件：什么时候停止。
3. 递归转移：如何把当前问题变成更小问题。

## 示例：阶乘

```text
factorial(n) = n * factorial(n - 1)
factorial(0) = 1
```

## 示例：树的遍历

树天然适合递归，因为每棵子树结构相同。

```text
遍历当前节点
递归遍历左子树
递归遍历右子树
```

## 易错点

- 没有终止条件导致无限递归。
- 递归参数没有向终止条件靠近。
- 只关注代码执行顺序，忽略函数语义。

## 递归与回溯

递归负责进入下一层问题，回溯负责撤销当前选择并尝试其他选择。

## C++刷题衔接

- 先会：函数调用、返回值、引用传参、递归终止条件。
- 开始写题：学完函数后可以写简单递归，学完指针和结构体后写树与链表。
- Hot 100 对应：二叉树最大深度、翻转二叉树、对称二叉树、全排列、子集。
- 写题要求：每个递归都写清楚“当前层做什么、何时返回、返回给上一层什么”。
