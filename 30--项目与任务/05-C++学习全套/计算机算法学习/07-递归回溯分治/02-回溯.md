# 回溯

回溯是一种系统搜索所有可能解的方法，本质是深度优先搜索。

## 通用框架

```text
选择当前候选
递归进入下一层
撤销当前选择
```

## 常见题型

- 子集
- 组合
- 排列
- N 皇后
- 数独
- 括号生成

## 剪枝

剪枝是在搜索过程中提前放弃不可能产生答案的分支。

常见剪枝：

- 剩余元素不足。
- 当前和已经超过目标。
- 重复元素跳过。

## 易错点

- 忘记撤销选择。
- 结果保存的是引用，后续被修改。
- 去重条件写错。

## C++刷题衔接

- 先会：递归、引用传参、`vector` 作为路径、撤销选择。
- 开始写题：学完递归后可以先做排列组合，学完字符串后做括号生成。
- Hot 100 对应：全排列、子集、组合总和、括号生成、单词搜索。
- 写题要求：固定模板是“选择 -> 递归 -> 撤销选择”，不要忘记去重条件。
