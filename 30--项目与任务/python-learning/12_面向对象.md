# 面向对象

## 类与对象

```python
class User:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"你好，我是 {self.name}"

user = User("Alice", 18)
print(user.greet())
```

## 实例属性与方法

`self` 表示当前对象本身。

```python
class Counter:
    def __init__(self):
        self.value = 0

    def increment(self):
        self.value += 1
```

## 类属性

```python
class Person:
    species = "Human"
```

## 继承

```python
class Animal:
    def speak(self):
        return "sound"

class Dog(Animal):
    def speak(self):
        return "wang"
```

## 封装

Python 中以下划线表示内部属性：

```python
class Account:
    def __init__(self):
        self._balance = 0
```
