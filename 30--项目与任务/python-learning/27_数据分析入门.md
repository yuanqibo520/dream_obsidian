# 数据分析入门

## 常用库

- NumPy：数值计算。
- pandas：表格数据处理。
- matplotlib：绘图。

## 安装

```bash
pip install numpy pandas matplotlib
```

## pandas 读取 CSV

```python
import pandas as pd

df = pd.read_csv("data.csv")
print(df.head())
print(df.info())
```

## 基础统计

```python
print(df.describe())
print(df["price"].mean())
print(df["price"].max())
```

## 筛选数据

```python
expensive = df[df["price"] > 100]
print(expensive)
```

## 分组统计

```python
result = df.groupby("category")["price"].mean()
print(result)
```

## 绘图

```python
import matplotlib.pyplot as plt

df["price"].plot(kind="hist")
plt.show()
```
