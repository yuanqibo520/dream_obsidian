# 数据类型

## 常见类型

| 类型 | 示例 | 说明 |
|---|---|---|
| `int` | `10` | 整数 |
| `float` | `3.14` | 浮点数 |
| `str` | `"hello"` | 字符串 |
| `bool` | `True` | 布尔值 |
| `NoneType` | `None` | 空值 |
| `list` | `[1, 2]` | 列表 |
| `tuple` | `(1, 2)` | 元组 |
| `dict` | `{"a": 1}` | 字典 |
| `set` | `{1, 2}` | 集合 |

## 查看类型

```python
x = 10
print(type(x))
```

## 类型转换

```python
age_text = "18"
age = int(age_text)

price = float("19.9")
message = str(123)
```

## 布尔判断

以下值通常被视为 `False`：

- `False`
- `None`
- `0`
- `""`
- `[]`
- `{}`
- `set()`

```python
if "":
    print("不会执行")
```

## 可变与不可变

不可变类型：`int`、`float`、`str`、`tuple`。

可变类型：`list`、`dict`、`set`。
