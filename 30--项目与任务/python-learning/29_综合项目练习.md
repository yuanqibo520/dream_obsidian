# 综合项目练习

## 项目一：命令行 Todo 工具

功能：

- 添加任务
- 查看任务
- 完成任务
- 删除任务
- 数据保存到 JSON 文件

涉及知识：

- 文件操作
- JSON
- argparse
- 函数拆分
- 异常处理

## 项目二：文件整理器

功能：

- 扫描下载目录
- 按扩展名分类文件
- 生成整理日志

涉及知识：

- pathlib
- shutil
- logging
- 自动化脚本

## 项目三：天气查询工具

功能：

- 接收城市名
- 请求天气 API
- 格式化输出
- 处理网络异常

涉及知识：

- requests
- JSON
- 异常处理
- 命令行参数

## 项目四：学生成绩管理系统

功能：

- 添加学生
- 修改成绩
- 查询学生
- 统计平均分
- 保存到 SQLite

涉及知识：

- 面向对象
- sqlite3
- SQL
- 测试

## 项目五：简单 Web API

功能：

- 用户列表接口
- 添加用户接口
- 查询用户接口
- 删除用户接口

涉及知识：

- Flask 或 FastAPI
- HTTP
- JSON
- REST API

## 项目完成标准

- 有 README。
- 能正常运行。
- 有基本错误处理。
- 至少有 3 个测试。
- 代码按模块拆分。
