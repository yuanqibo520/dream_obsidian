# 正则表达式

## re 模块

```python
import re

text = "我的手机号是 13800138000"
match = re.search(r"\d{11}", text)

if match:
    print(match.group())
```

## 常用符号

| 符号 | 含义 |
|---|---|
| `.` | 任意字符 |
| `\d` | 数字 |
| `\w` | 字母、数字、下划线 |
| `\s` | 空白字符 |
| `*` | 0 次或多次 |
| `+` | 1 次或多次 |
| `?` | 0 次或 1 次 |
| `{n}` | n 次 |

## findall

```python
text = "a1 b22 c333"
print(re.findall(r"\d+", text))
```

## 替换

```python
text = "hello 123"
new_text = re.sub(r"\d+", "***", text)
print(new_text)
```

## 分组

```python
match = re.search(r"(\d{4})-(\d{2})-(\d{2})", "2026-05-30")
if match:
    print(match.group(1))
```
