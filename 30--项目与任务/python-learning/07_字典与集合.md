# 字典与集合

## 字典

字典使用键值对存储数据。

```python
user = {
    "name": "Alice",
    "age": 18,
}

print(user["name"])
```

## 常用操作

```python
user["city"] = "Shanghai"
user["age"] = 19

print(user.get("email", "未填写"))

for key, value in user.items():
    print(key, value)
```

## 字典推导式

```python
squares = {x: x * x for x in range(5)}
```

## 集合

集合是无序、不重复的数据容器。

```python
tags = {"python", "web", "python"}
print(tags)
```

## 集合运算

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)
print(a & b)
print(a - b)
```
