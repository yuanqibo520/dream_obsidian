# 函数

## 定义函数

```python
def greet(name):
    return "你好，" + name

print(greet("Alice"))
```

## 参数

### 位置参数

```python
def add(a, b):
    return a + b
```

### 默认参数

```python
def greet(name, prefix="你好"):
    return prefix + "，" + name
```

### 关键字参数

```python
print(greet(name="Bob", prefix="早上好"))
```

## 返回值

函数可以返回一个或多个值：

```python
def get_user():
    return "Alice", 18

name, age = get_user()
```

## 作用域

```python
x = 10

def show():
    x = 20
    print(x)

show()
print(x)
```

函数内部的 `x` 和外部的 `x` 是不同变量。

## Lambda 表达式

```python
square = lambda x: x * x
print(square(5))
```
