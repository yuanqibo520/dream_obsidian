# 类型注解

## 基本写法

```python
name: str = "Alice"
age: int = 18
price: float = 19.9
is_active: bool = True
```

## 函数注解

```python
def add(a: int, b: int) -> int:
    return a + b
```

## 容器类型

```python
names: list[str] = ["Alice", "Bob"]
scores: dict[str, int] = {"Alice": 90}
point: tuple[int, int] = (10, 20)
```

## Optional

```python
from typing import Optional

def find_user(user_id: int) -> Optional[str]:
    if user_id == 1:
        return "Alice"
    return None
```

在 Python 3.10+ 中也可以写：

```python
def find_user(user_id: int) -> str | None:
    return None
```

## 类型注解的作用

- 提升代码可读性。
- 帮助编辑器提示。
- 配合 mypy、pyright 做静态检查。
