# 环境与运行方式

## Python 是什么

Python 是一种解释型、跨平台、语法简洁的通用编程语言，常用于自动化、Web 开发、数据分析、人工智能、脚本工具和后端服务。

## 安装 Python

建议安装 Python 3.11 或更新版本。安装后在终端检查：

```bash
python --version
```

有些系统使用：

```bash
python3 --version
```

## 运行 Python 的方式

### 交互式运行

```bash
python
```

进入后可以直接输入表达式：

```python
print("Hello Python")
```

### 运行脚本文件

创建 `hello.py`：

```python
print("Hello Python")
```

运行：

```bash
python hello.py
```

## 推荐编辑器

- VS Code
- PyCharm
- Jupyter Notebook

## 第一个程序

```python
name = input("请输入你的名字：")
print("你好，" + name)
```

## 常见问题

- `python` 命令找不到：检查是否安装并加入 PATH。
- 中文乱码：保存文件时使用 UTF-8。
- 版本混乱：使用虚拟环境隔离项目。
