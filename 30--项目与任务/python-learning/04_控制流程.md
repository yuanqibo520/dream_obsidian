# 控制流程

## if 判断

```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")
```

## 比较运算符

```python
print(3 > 2)
print(3 >= 3)
print(3 == 3)
print(3 != 4)
```

## 逻辑运算符

```python
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("可以入场")
```

## for 循环

```python
for i in range(5):
    print(i)
```

遍历列表：

```python
names = ["Alice", "Bob", "Cindy"]

for name in names:
    print(name)
```

## while 循环

```python
count = 0

while count < 3:
    print(count)
    count += 1
```

## break 与 continue

```python
for i in range(10):
    if i == 3:
        continue
    if i == 8:
        break
    print(i)
```
