# 数据库入门

## SQLite

SQLite 是 Python 标准库自带的轻量级数据库。

## 连接数据库

```python
import sqlite3

conn = sqlite3.connect("app.db")
cursor = conn.cursor()
```

## 创建表

```python
cursor.execute("""
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    age INTEGER
)
""")
conn.commit()
```

## 插入数据

```python
cursor.execute(
    "INSERT INTO users (name, age) VALUES (?, ?)",
    ("Alice", 18),
)
conn.commit()
```

## 查询数据

```python
cursor.execute("SELECT id, name, age FROM users")
rows = cursor.fetchall()

for row in rows:
    print(row)
```

## 关闭连接

```python
conn.close()
```
