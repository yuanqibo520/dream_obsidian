# Python 学习路线

## 学习目标

这套资料按模块拆分，每个文件覆盖一个核心知识点。建议从基础语法开始，逐步进入工程化、自动化、Web、数据分析和项目实践。

## 推荐顺序

1. 环境与运行方式
2. 基础语法
3. 数据类型
4. 控制流程
5. 函数
6. 常用数据结构
7. 字符串
8. 文件操作
9. 异常处理
10. 模块与包
11. 面向对象
12. 迭代器与生成器
13. 装饰器
14. 类型注解
15. 虚拟环境与依赖管理
16. 常用标准库
17. 测试
18. 调试与日志
19. 正则表达式
20. JSON 与数据交换
21. 网络请求
22. 数据库
23. 并发编程
24. 自动化脚本
25. Web 开发入门
26. 数据分析入门
27. 项目结构与最佳实践

## 学习方法

- 每学完一个文件，手写其中的示例。
- 不要只看语法，要主动修改代码并观察结果。
- 每个模块至少完成一个小练习。
- 学完基础后，尽快做项目，例如命令行记账工具、文件整理脚本、简单 Web API。

## 推荐练习项目

- BMI 计算器
- 猜数字游戏
- Todo 命令行工具
- 批量重命名文件脚本
- 天气查询脚本
- Flask/FastAPI 简单接口
- CSV 数据分析报表
