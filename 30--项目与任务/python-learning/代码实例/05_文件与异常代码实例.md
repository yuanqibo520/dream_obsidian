# 文件与异常代码实例

## 示例 1：写入日记

```python
from pathlib import Path

path = Path("diary.txt")
content = input("今天学到了什么：")

path.write_text(content, encoding="utf-8")
print("保存成功")
```

## 示例 2：读取文件并统计行数

```python
from pathlib import Path

path = Path("data.txt")

if path.exists():
    lines = path.read_text(encoding="utf-8").splitlines()
    print("行数：", len(lines))
else:
    print("文件不存在")
```

## 示例 3：安全读取数字

```python
try:
    number = int(input("请输入数字："))
except ValueError:
    print("输入错误，请输入整数")
else:
    print("你输入的是：", number)
```

## 示例 4：读取 JSON 配置

```python
import json
from pathlib import Path

config_path = Path("config.json")

try:
    config = json.loads(config_path.read_text(encoding="utf-8"))
except FileNotFoundError:
    config = {"debug": False}
except json.JSONDecodeError:
    print("配置文件格式错误")
    config = {"debug": False}

print(config)
```

## 示例 5：批量列出文件

```python
from pathlib import Path

folder = Path(".")

for path in folder.iterdir():
    if path.is_file():
        print(path.name, path.stat().st_size)
```

## 示例 6：复制文件

```python
from pathlib import Path
import shutil

source = Path("source.txt")
target = Path("backup.txt")

if source.exists():
    shutil.copy(source, target)
    print("复制完成")
else:
    print("源文件不存在")
```
