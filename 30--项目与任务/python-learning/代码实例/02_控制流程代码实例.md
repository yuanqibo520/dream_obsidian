# 控制流程代码实例

## 示例 1：成绩等级判断

```python
score = int(input("请输入成绩："))

if score >= 90:
    level = "A"
elif score >= 80:
    level = "B"
elif score >= 60:
    level = "C"
else:
    level = "D"

print(f"等级：{level}")
```

## 示例 2：判断闰年

```python
year = int(input("请输入年份："))

if (year % 4 == 0 and year % 100 != 0) or year % 400 == 0:
    print("闰年")
else:
    print("平年")
```

## 示例 3：累加求和

```python
total = 0

for number in range(1, 101):
    total += number

print(total)
```

## 示例 4：九九乘法表

```python
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{j}x{i}={i * j}", end="\t")
    print()
```

## 示例 5：猜数字游戏

```python
import random

answer = random.randint(1, 100)

while True:
    guess = int(input("猜一个 1 到 100 的数字："))

    if guess > answer:
        print("太大了")
    elif guess < answer:
        print("太小了")
    else:
        print("猜对了")
        break
```

## 示例 6：菜单循环

```python
while True:
    print("1. 添加")
    print("2. 查看")
    print("0. 退出")

    choice = input("请选择：")

    if choice == "1":
        print("执行添加")
    elif choice == "2":
        print("执行查看")
    elif choice == "0":
        print("退出程序")
        break
    else:
        print("无效选择")
```
