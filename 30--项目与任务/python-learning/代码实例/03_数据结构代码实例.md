# 数据结构代码实例

## 示例 1：列表增删改查

```python
tasks = ["学习 Python", "写作业"]

tasks.append("复习")
tasks[1] = "完成每日作业"
tasks.remove("复习")

for index, task in enumerate(tasks, start=1):
    print(index, task)
```

## 示例 2：列表去重并保持顺序

```python
numbers = [1, 2, 2, 3, 1, 4]
result = []

for number in numbers:
    if number not in result:
        result.append(number)

print(result)
```

## 示例 3：统计成绩

```python
scores = [88, 92, 76, 65, 99]

print("最高分：", max(scores))
print("最低分：", min(scores))
print("平均分：", sum(scores) / len(scores))
```

## 示例 4：字典保存用户

```python
user = {
    "name": "Alice",
    "age": 18,
    "skills": ["Python", "SQL"],
}

user["city"] = "Shanghai"

print(user["name"])
print(user.get("email", "未填写"))
```

## 示例 5：统计字符出现次数

```python
text = "banana"
counter = {}

for char in text:
    counter[char] = counter.get(char, 0) + 1

print(counter)
```

## 示例 6：集合找共同爱好

```python
alice = {"电影", "音乐", "编程"}
bob = {"编程", "篮球", "音乐"}

print("共同爱好：", alice & bob)
print("所有爱好：", alice | bob)
```

## 示例 7：学生成绩表

```python
students = [
    {"name": "Alice", "score": 90},
    {"name": "Bob", "score": 75},
    {"name": "Cindy", "score": 88},
]

for student in students:
    if student["score"] >= 80:
        print(student["name"], "优秀")
```
