# 基础语法代码实例

## 示例 1：个人信息卡片

```python
name = "张三"
age = 20
city = "上海"

print("姓名：", name)
print("年龄：", age)
print("城市：", city)
```

## 示例 2：输入并计算年龄

```python
birth_year = int(input("请输入出生年份："))
current_year = 2026
age = current_year - birth_year

print(f"你今年大约 {age} 岁")
```

## 示例 3：计算圆面积

```python
radius = float(input("请输入圆半径："))
pi = 3.14159
area = pi * radius ** 2

print(f"圆面积是：{area:.2f}")
```

## 示例 4：温度转换

```python
celsius = float(input("请输入摄氏度："))
fahrenheit = celsius * 9 / 5 + 32

print(f"{celsius} 摄氏度 = {fahrenheit:.1f} 华氏度")
```

## 示例 5：简单收银台

```python
price = float(input("商品单价："))
count = int(input("购买数量："))
discount = float(input("折扣，例如 0.8 表示八折："))

total = price * count * discount
print(f"应付金额：{total:.2f} 元")
```

## 示例 6：交换变量

```python
a = 10
b = 20

a, b = b, a

print(a)
print(b)
```
