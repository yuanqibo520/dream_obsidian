# 并发编程

## 并发的用途

并发适合处理网络请求、文件 IO、批量任务等等待时间较多的场景。

## threading

```python
import threading

def worker(name):
    print("worker", name)

thread = threading.Thread(target=worker, args=("A",))
thread.start()
thread.join()
```

## concurrent.futures

```python
from concurrent.futures import ThreadPoolExecutor

def task(x):
    return x * x

with ThreadPoolExecutor(max_workers=4) as executor:
    results = executor.map(task, range(10))
    print(list(results))
```

## multiprocessing

适合 CPU 密集型任务。

```python
from multiprocessing import Pool

def square(x):
    return x * x

if __name__ == "__main__":
    with Pool(4) as pool:
        print(pool.map(square, range(10)))
```

## asyncio

```python
import asyncio

async def hello():
    await asyncio.sleep(1)
    print("hello")

asyncio.run(hello())
```

## 选择建议

- 网络 IO：`asyncio` 或线程池。
- 文件 IO：线程池。
- CPU 密集：多进程。
