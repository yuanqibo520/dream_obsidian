# 自动化脚本

## 自动化能做什么

- 批量重命名文件
- 整理文件夹
- 读取 Excel/CSV
- 定时运行任务
- 调用 API
- 生成报告

## 批量重命名

```python
from pathlib import Path

folder = Path("files")

for index, path in enumerate(folder.glob("*.txt"), start=1):
    new_name = folder / f"file_{index}.txt"
    path.rename(new_name)
```

## 文件分类

```python
from pathlib import Path
import shutil

folder = Path("downloads")

for path in folder.iterdir():
    if path.is_file():
        target_folder = folder / path.suffix.lstrip(".")
        target_folder.mkdir(exist_ok=True)
        shutil.move(str(path), str(target_folder / path.name))
```
