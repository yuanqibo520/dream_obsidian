# 字符串

## 创建字符串

```python
name = "Alice"
message = 'Hello'
multi_line = """多行字符串"""
```

## 字符串格式化

推荐使用 f-string：

```python
name = "Alice"
age = 18
print(f"{name} 今年 {age} 岁")
```

## 常用方法

```python
text = " hello python "

print(text.strip())
print(text.upper())
print(text.lower())
print(text.replace("python", "world"))
print(text.split())
```

## 判断方法

```python
print("123".isdigit())
print("abc".isalpha())
print("abc123".isalnum())
```

## 切片

```python
text = "python"
print(text[0])
print(text[-1])
print(text[1:4])
print(text[::-1])
```
