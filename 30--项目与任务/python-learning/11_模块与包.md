# 模块与包

## 模块

一个 `.py` 文件就是一个模块。

`math_utils.py`：

```python
def add(a, b):
    return a + b
```

使用模块：

```python
import math_utils

print(math_utils.add(1, 2))
```

## from import

```python
from math_utils import add

print(add(1, 2))
```

## 标准库模块

```python
import math
import random
from datetime import datetime

print(math.sqrt(16))
print(random.randint(1, 10))
print(datetime.now())
```

## 包

包是包含 `__init__.py` 的文件夹。

```text
my_package/
  __init__.py
  utils.py
```

## `__name__ == "__main__"`

```python
def main():
    print("程序入口")

if __name__ == "__main__":
    main()
```

这样可以避免模块被导入时自动执行入口逻辑。
