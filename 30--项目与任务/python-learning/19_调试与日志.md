# 调试与日志

## print 调试

```python
value = 10
print("value =", value)
```

适合简单场景，但大型项目中不建议到处使用。

## pdb 调试

```python
import pdb

x = 10
pdb.set_trace()
y = x + 1
print(y)
```

## logging

```python
import logging

logging.basicConfig(level=logging.INFO)

logging.info("程序开始")
logging.warning("这是警告")
logging.error("这是错误")
```

## 日志级别

- `DEBUG`
- `INFO`
- `WARNING`
- `ERROR`
- `CRITICAL`

## 写入文件

```python
import logging

logging.basicConfig(
    filename="app.log",
    level=logging.INFO,
    format="%(asctime)s %(levelname)s %(message)s",
)

logging.info("hello")
```
