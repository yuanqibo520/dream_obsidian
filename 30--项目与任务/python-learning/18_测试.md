# 测试

## 为什么写测试

测试可以帮助你确认代码行为是否符合预期，减少修改代码后引入新问题的风险。

## 使用 assert

```python
def add(a, b):
    return a + b

assert add(1, 2) == 3
```

## unittest

```python
import unittest

def add(a, b):
    return a + b

class TestMath(unittest.TestCase):
    def test_add(self):
        self.assertEqual(add(1, 2), 3)

if __name__ == "__main__":
    unittest.main()
```

## pytest

安装：

```bash
pip install pytest
```

测试文件：

```python
def add(a, b):
    return a + b

def test_add():
    assert add(1, 2) == 3
```

运行：

```bash
pytest
```

## 测试原则

- 测试正常输入。
- 测试边界条件。
- 测试异常情况。
- 测试应该小而清晰。

## 练习

1. 为计算器函数写测试。
2. 为字符串反转函数写测试。
3. 测试除以零是否抛出异常。
