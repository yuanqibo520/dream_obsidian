# Day 14：模块与包

## 学习目标

- 能把代码拆成模块。
- 理解 `import`。

## 编码任务

创建一个小包：

```text
calculator/
  __init__.py
  basic.py
  advanced.py
main.py
```

要求：

1. `basic.py` 放加减乘除。
2. `advanced.py` 放平方、开方。
3. `main.py` 导入并调用。

## 加分任务

在 `main.py` 中加入菜单选择功能。

## 自测问题

- 一个 `.py` 文件为什么可以叫模块？
- `if __name__ == "__main__"` 有什么用？
