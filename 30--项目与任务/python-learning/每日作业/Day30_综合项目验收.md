# Day 30：综合项目验收

## 学习目标

- 独立完成一个小项目。
- 检查一个月学习成果。

## 项目选题

任选一个：

1. 命令行 Todo 工具
2. 学生成绩管理系统
3. 文件整理器
4. 简易记账本
5. 天气查询工具

## 必做要求

1. 至少拆分 3 个函数。
2. 至少使用 1 个文件保存数据。
3. 至少处理 2 种异常。
4. 有 README。
5. 有 3 个测试用例或手动测试记录。

## 验收标准

满分 100 分：

- 功能完整：30 分
- 代码结构清晰：20 分
- 异常处理：15 分
- 数据持久化：15 分
- 测试记录：10 分
- README：10 分

## 自测问题

- 项目中最难的部分是什么？
- 哪段代码最值得重构？
- 如果继续开发，你会加什么功能？

## 提交清单

- 完整项目文件夹
- README
- 运行截图或文字输出
- 30 天学习总结
