# Day 01：环境、输入输出、变量

## 学习目标

- 能运行 Python 文件。
- 能使用变量保存数据。
- 能使用 `input()` 和 `print()`。

## 编码任务

写一个 `profile.py`，实现：

1. 输入姓名、年龄、城市。
2. 输出一张个人信息卡片。
3. 输出“明年你将 X 岁”。

## 加分任务

让输出格式更整齐，例如：

```text
======== 个人信息 ========
姓名：Alice
年龄：18
城市：Shanghai
明年年龄：19
```

## 自测问题

- `input()` 得到的数据默认是什么类型？
- 为什么年龄需要用 `int()` 转换？
