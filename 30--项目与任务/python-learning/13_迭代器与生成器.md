# 迭代器与生成器

## 可迭代对象

可以被 `for` 遍历的对象叫可迭代对象，例如列表、字符串、字典。

```python
for char in "python":
    print(char)
```

## 迭代器

迭代器实现了 `__iter__` 和 `__next__`。

```python
items = iter([1, 2, 3])
print(next(items))
print(next(items))
```

## 生成器函数

使用 `yield` 返回值的函数叫生成器函数。

```python
def count_up_to(n):
    current = 1
    while current <= n:
        yield current
        current += 1

for number in count_up_to(3):
    print(number)
```

## 生成器表达式

```python
squares = (x * x for x in range(10))
```

## 优点

- 节省内存。
- 适合处理大文件、大数据流。
- 可以按需生成数据。
