# 基础语法

## 注释

单行注释使用 `#`：

```python
# 这是注释
print("hello")
```

多行说明通常使用三引号字符串：

```python
"""
这里可以写多行说明。
"""
```

## 缩进

Python 使用缩进表示代码块，通常使用 4 个空格。

```python
if True:
    print("缩进内的代码属于 if")
```

## 变量

变量不需要提前声明：

```python
name = "Alice"
age = 18
height = 1.68
```

## 命名规范

- 变量和函数：`snake_case`
- 类名：`PascalCase`
- 常量：`UPPER_CASE`

```python
user_name = "Bob"
MAX_RETRY = 3
```

## 输入与输出

```python
name = input("你的名字：")
print("你好", name)
```

## 基础运算

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
```
