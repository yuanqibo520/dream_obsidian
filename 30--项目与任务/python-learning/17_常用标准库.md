# 常用标准库

## pathlib

```python
from pathlib import Path

path = Path("data.txt")
print(path.exists())
```

## datetime

```python
from datetime import datetime, timedelta

now = datetime.now()
tomorrow = now + timedelta(days=1)
print(now, tomorrow)
```

## random

```python
import random

print(random.randint(1, 10))
print(random.choice(["a", "b", "c"]))
```

## os

```python
import os

print(os.getcwd())
print(os.environ.get("PATH"))
```

## sys

```python
import sys

print(sys.argv)
print(sys.version)
```

## collections

```python
from collections import Counter, defaultdict

print(Counter("banana"))

groups = defaultdict(list)
groups["fruit"].append("apple")
```

## itertools

```python
import itertools

for item in itertools.product([1, 2], ["a", "b"]):
    print(item)
```

## 练习

1. 使用 `Counter` 统计单词频率。
2. 使用 `pathlib` 批量列出文件。
3. 使用 `datetime` 计算 100 天后的日期。
