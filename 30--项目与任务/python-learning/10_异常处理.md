# 异常处理

## 什么是异常

异常是程序运行时出现的错误，例如文件不存在、除以零、类型不匹配。

## try except

```python
try:
    number = int(input("请输入数字："))
    print(10 / number)
except ValueError:
    print("输入不是数字")
except ZeroDivisionError:
    print("不能除以 0")
```

## else 与 finally

```python
try:
    result = 10 / 2
except ZeroDivisionError:
    print("错误")
else:
    print("没有异常时执行")
finally:
    print("无论是否异常都会执行")
```

## 主动抛出异常

```python
def set_age(age):
    if age < 0:
        raise ValueError("年龄不能为负数")
    return age
```

## 自定义异常

```python
class BusinessError(Exception):
    pass

raise BusinessError("业务错误")
```
