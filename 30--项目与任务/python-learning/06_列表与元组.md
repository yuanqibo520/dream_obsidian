# 列表与元组

## 列表

列表是有序、可变的数据容器。

```python
numbers = [1, 2, 3]
numbers.append(4)
numbers.remove(2)
print(numbers)
```

## 常用操作

```python
items = ["a", "b", "c"]

print(items[0])
print(items[-1])
print(items[1:3])
print(len(items))
```

## 列表推导式

```python
squares = [x * x for x in range(10)]
evens = [x for x in range(10) if x % 2 == 0]
```

## 排序

```python
numbers = [3, 1, 2]
numbers.sort()
print(numbers)

sorted_numbers = sorted(numbers, reverse=True)
```

## 元组

元组是有序、不可变的数据容器。

```python
point = (10, 20)
x, y = point
```

## 什么时候用元组

- 数据不希望被修改。
- 表示固定结构，例如坐标、日期片段。
- 函数返回多个值。
