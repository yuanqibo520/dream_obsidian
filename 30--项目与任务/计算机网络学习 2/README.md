# 计算机网络学习

这是一套按模块拆分的计算机网络学习笔记。每个 Markdown 文件覆盖一个核心知识点，适合按顺序学习，也适合在复习、面试、排错时快速查阅。

## 推荐学习顺序

1. 先读 `00-学习路线`
2. 再读 `01-网络基础`
3. 按自底向上的顺序学习：物理层、数据链路层、网络层、传输层、应用层
4. 最后补充网络安全、工程排错和面试复习

## 模块目录

- `00-学习路线`：学习地图、常用术语、覆盖检查
- `01-网络基础`：分层模型、封装、地址体系、交换方式、性能指标
- `02-物理层`：信号、带宽、传输介质、编码调制、多路复用、信道容量
- `03-数据链路层`：以太网、MAC、交换机、VLAN、ARP、差错检测、介质访问控制
- `04-网络层`：IP、子网、路由、ICMP、NAT、IPv6、动态路由
- `05-传输层`：TCP、UDP、可靠传输、拥塞控制、状态机、QUIC
- `06-应用层`：DNS、HTTP、HTTPS、WebSocket、邮件、DHCP、缓存、CDN
- `07-网络安全`：TLS、防火墙、常见攻击、PKI、零信任
- `08-网络工程与排错`：抓包、命令、性能指标、SNMP
- `09-面试与复习`：高频题、速查表
