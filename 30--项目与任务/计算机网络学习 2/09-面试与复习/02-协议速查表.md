# 协议速查表

| 协议 | 层次 | 默认端口 | 作用 |
|---|---:|---:|---|
| Ethernet | 数据链路层 | 无 | 局域网帧传输 |
| ARP | 数据链路层/网络层之间 | 无 | IPv4 地址解析为 MAC |
| IP | 网络层 | 无 | 跨网络寻址和转发 |
| ICMP | 网络层 | 无 | 错误报告和诊断 |
| TCP | 传输层 | 无 | 可靠字节流传输 |
| UDP | 传输层 | 无 | 低开销数据报传输 |
| DNS | 应用层 | 53 | 域名解析 |
| HTTP | 应用层 | 80 | Web 通信 |
| HTTPS | 应用层 | 443 | 加密 Web 通信 |
| SSH | 应用层 | 22 | 安全远程登录 |
| SMTP | 应用层 | 25 | 邮件发送 |
| IMAP | 应用层 | 143 | 邮件同步 |
| POP3 | 应用层 | 110 | 邮件下载 |
| DHCP | 应用层 | 67/68 | 自动分配网络配置 |
| BGP | 应用层/路由协议 | 179 | AS 间路由 |
