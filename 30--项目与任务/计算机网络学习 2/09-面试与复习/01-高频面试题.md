# 高频面试题

## 输入 URL 到页面展示发生了什么

浏览器解析 URL、查询缓存、DNS 解析、TCP 握手、TLS 握手、HTTP 请求、服务器响应、浏览器渲染。

## TCP 和 UDP 的区别

TCP 可靠、有连接、按序交付、开销较大。UDP 无连接、不保证可靠、开销小、延迟低。

## 三次握手为什么是三次

三次握手可以确认双方发送和接收能力，并同步初始序列号。

## HTTP 和 HTTPS 的区别

HTTPS 在 HTTP 基础上加入 TLS，提供加密、完整性保护和身份认证。

## 交换机和路由器区别

交换机根据 MAC 地址转发帧，路由器根据 IP 地址转发数据包。
