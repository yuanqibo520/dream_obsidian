# 常用网络命令

## Windows

```powershell
ipconfig /all
ping 8.8.8.8
tracert example.com
nslookup example.com
netstat -ano
route print
arp -a
```

## Linux/macOS

```bash
ip addr
ip route
ping 8.8.8.8
traceroute example.com
dig example.com
ss -tunlp
curl -v https://example.com
```

## 排错顺序

先检查本机 IP、网关、DNS，再检查网关、公网 IP、DNS、目标端口和抓包结果。
