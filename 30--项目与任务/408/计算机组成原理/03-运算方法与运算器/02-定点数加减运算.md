# 定点数加减运算

## 补码加法

补码加法直接按二进制加法进行，符号位参与运算，最高位进位丢弃。

```text
[x + y]补 = [x]补 + [y]补
```

## 补码减法

减法可转化为加法：

```text
[x - y]补 = [x]补 + [-y]补
```

## 溢出条件

有符号数加法中：

- 正数 + 正数 = 负数，溢出。
- 负数 + 负数 = 正数，溢出。
- 异号相加不会溢出。

## 双符号位判断

双符号位又称变形补码：

- `00`：正数，无溢出。
- `11`：负数，无溢出。
- `01`：正溢出。
- `10`：负溢出。

## 易错点

- 无符号数关注进位 CF。
- 有符号补码数关注溢出 OF。
- 最高位进位不等于有符号溢出。
