# 常见计算题

## 进制转换

重点掌握：

- 二进制、八进制、十进制、十六进制互转。
- 小数部分乘基取整。
- 负数补码表示。

## 补码范围

n 位补码表示范围：

```text
-2^(n-1) 到 2^(n-1)-1
```

## Cache 地址划分

通常按以下字段划分：

```text
主存地址 = 标记 Tag + Cache 索引 Index + 块内地址 Offset
```

块内地址位数：

```text
log2(块大小)
```

行号或组号位数：

```text
log2(Cache 行数或组数)
```

## 平均访问时间

```text
AMAT = 命中时间 + 失效率 × 失效代价
```

## 磁盘访问时间

```text
访问时间 = 寻道时间 + 平均旋转延迟 + 传输时间
平均旋转延迟 = 旋转一周时间 / 2
```

## 流水线时间

k 段流水线执行 n 条指令：

```text
T = (k + n - 1) × 时钟周期
```

## 总线带宽

```text
总线带宽 = 总线宽度 × 总线频率
```

注意 bit 和 Byte 的换算：1 Byte = 8 bit。
