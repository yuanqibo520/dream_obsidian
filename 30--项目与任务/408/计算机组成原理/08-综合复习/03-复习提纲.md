# 复习提纲

## 第一轮：建立框架

- 计算机系统层次结构。
- 冯诺依曼结构。
- CPU、主存、I/O 的关系。
- 指令执行的基本过程。

## 第二轮：掌握计算

- 进制转换。
- 补码表示和溢出判断。
- 浮点数格式。
- Cache 地址划分与平均访问时间。
- 磁盘访问时间。
- 流水线执行时间。
- 总线带宽。

## 第三轮：专题突破

- 数据表示。
- 运算器。
- 存储系统。
- 指令系统。
- CPU 数据通路与控制器。
- I/O 系统。

## 高频简答题

1. 冯诺依曼结构的基本特点。
2. Cache 的工作原理和地址映射方式。
3. 虚拟存储器的作用和地址转换过程。
4. RISC 与 CISC 的区别。
5. 中断处理过程。
6. DMA 工作过程。
7. 流水线冲突及解决方法。

## 考前检查

- 能否手算补码和溢出。
- 能否画出指令周期基本流程。
- 能否计算 Cache 字段位数。
- 能否区分 MAR、MDR、PC、IR。
- 能否说清中断和 DMA 的区别。
