# 指令周期

## 定义

指令周期是 CPU 从取出一条指令到执行完这条指令所需的全部时间。

## 常见阶段

1. 取指周期。
2. 间址周期。
3. 执行周期。
4. 中断周期。

并非每条指令都需要所有阶段。

## 取指周期

典型微操作：

```text
MAR <- PC
MDR <- M[MAR]
IR <- MDR
PC <- PC + 1
```

## 间址周期

当指令采用间接寻址时，需要根据地址字段再访存得到有效地址。

## 执行周期

根据指令类型执行运算、访存或转移。

## 中断周期

保存断点，转入中断服务程序。

## 常考点

取指周期通常所有指令都有，间址周期和中断周期不一定出现。
