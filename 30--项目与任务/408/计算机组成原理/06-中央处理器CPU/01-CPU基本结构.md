# CPU 基本结构

## CPU 组成

CPU 通常由运算器、控制器和寄存器组组成。

## 运算器

负责算术和逻辑运算，核心部件是 ALU。

## 控制器

负责取指、译码、产生控制信号，协调各部件工作。

## 寄存器

常见寄存器：

- PC：程序计数器，保存下一条指令地址。
- IR：指令寄存器，保存当前指令。
- MAR：存储器地址寄存器，保存访存地址。
- MDR：存储器数据寄存器，保存访存数据。
- ACC：累加器。
- PSW：程序状态字，保存标志位和状态信息。

## CPU 工作核心

CPU 的本质工作是不断重复：

```text
取指 -> 译码 -> 执行
```

## 常考点

PC 保存的是下一条指令地址，IR 保存的是当前正在执行的指令。
