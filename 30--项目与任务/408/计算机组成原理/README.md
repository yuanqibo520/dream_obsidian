# 计算机组成原理学习总览

这套笔记按“模块 -> 知识点”拆分，一个文件覆盖一个核心知识点，适合用于期末复习、考研复习或系统补基础。

## 学习路径

1. 先理解计算机系统的层次结构和性能指标。
2. 学习数据表示，掌握补码、浮点数和校验码。
3. 进入运算器，理解 ALU、定点运算和浮点运算。
4. 学习存储系统，重点是 Cache、主存、虚拟存储器。
5. 学习指令系统，理解寻址方式、指令格式和 CISC/RISC。
6. 学习 CPU，重点是数据通路、控制器、流水线和中断。
7. 学习总线与 I/O，理解外设如何与主机交换数据。
8. 用综合题把各模块串起来。
