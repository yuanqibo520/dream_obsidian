# 高速缓冲存储器 Cache

## 作用

Cache 位于 CPU 和主存之间，用于缓解 CPU 与主存速度不匹配的问题。

## 基本单位

Cache 与主存之间以块为单位交换数据。块也叫 Cache 行。

## 地址映射方式

### 直接映射

每个主存块只能映射到一个 Cache 行。

优点：实现简单。

缺点：冲突率较高。

### 全相联映射

任意主存块可放入任意 Cache 行。

优点：冲突率低。

缺点：硬件复杂。

### 组相联映射

主存块映射到固定组，组内任意行可放置。

折中方案，最常见。

## 替换算法

- 随机替换。
- FIFO。
- LRU。

## 写策略

### 写直达

写 Cache 时同时写主存。

### 写回

只写 Cache，替换时再写回主存。

## 常考点

- 地址字段划分：标记、索引、块内地址。
- 平均访问时间计算。
- 写命中和写不命中的处理策略。
