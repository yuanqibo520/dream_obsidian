# HTTP 状态码

## 2xx

- 200 OK。
- 201 Created。
- 204 No Content。

## 3xx

- 301 Moved Permanently。
- 302 Found。
- 304 Not Modified。

## 4xx

- 400 Bad Request。
- 401 Unauthorized。
- 403 Forbidden。
- 404 Not Found。
- 429 Too Many Requests。

## 5xx

- 500 Internal Server Error。
- 502 Bad Gateway。
- 503 Service Unavailable。
- 504 Gateway Timeout。
