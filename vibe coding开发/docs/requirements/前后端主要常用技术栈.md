## 常见技术栈组合

### 1. 中小型 Web 项目

- 前端：Vue 3 + Vite + TypeScript + Element Plus
- 后端：Spring Boot + MySQL + Redis

### 2. React 项目

- 前端：React + Vite + TypeScript + Ant Design
- 后端：NestJS + PostgreSQL + Redis

### 3. Python 快速开发项目

- 前端：Vue / React
- 后端：FastAPI + PostgreSQL + Redis

### 4. 企业级管理系统

- 前端：Vue 3 + TypeScript + Ant Design Vue
- 后端：Spring Boot + MyBatis Plus + MySQL + Redis + RabbitMQ

### 5. 全栈 JavaScript 项目

- 前端：Next.js + TypeScript + Tailwind CSS
- 后端：Node.js / NestJS + Prisma + PostgreSQL

## 学习推荐路线

### 前端路线

1. HTML + CSS + JavaScript
2. TypeScript
3. Vue 或 React
4. Vite + 路由 + 状态管理
5. Axios / Fetch / TanStack Query
6. UI 组件库
7. 前端工程化
8. Next.js 或 Nuxt

### 后端路线

1. 一门后端语言：Java / Python / Node.js / Go
2. 一个主流框架：Spring Boot / FastAPI / NestJS / Gin
3. 数据库：MySQL / PostgreSQL
4. 缓存：Redis
5. 接口设计：RESTful API / GraphQL
6. 身份认证：JWT / OAuth2
7. 部署：Linux + Nginx + Docker
8. 进阶：消息队列、微服务、CI/CD、Kubernetes
