# 程序员能力成长地图

## 1. 技术栈知识点

技术栈知识点决定你能不能把功能做出来。

重点关注具体工具、语言、框架和工程能力：

- 编程语言：JavaScript、TypeScript、Python、Java、Go
- 前端技术：HTML、CSS、React、Vue、Next.js
- 后端技术：Node.js、Spring Boot、Django、FastAPI
- 数据库：MySQL、PostgreSQL、Redis、MongoDB
- 工程化：Git、Docker、CI/CD、自动化测试、部署
- 基础设施：Linux、Nginx、云服务器、对象存储、Kubernetes
- AI 工具：大模型 API、Prompt、Agent、RAG、向量数据库

## 2. 架构思维

架构思维决定你能不能把系统做大、做稳、做久。

重点关注系统如何组织、扩展和演进：

- 分层架构
- 模块化设计
- 领域划分
- 单体架构、微服务架构、Serverless 架构
- 高并发、高可用、可扩展性
- 缓存设计
- 消息队列与异步任务
- 数据一致性、事务、幂等性
- 日志、监控、链路追踪
- 技术选型与取舍

## 3. 设计能力

设计能力决定你能不能把需求做对、做清楚、做得可维护。

重点关注从需求到方案的转化能力：

- 需求拆解
- 业务流程设计
- 数据模型设计
- API 设计
- 页面交互设计
- 权限模型设计
- 状态流转设计
- 异常场景设计
- 边界条件设计
- 可维护性设计
- 可测试性设计

## 4. 软技能

软技能决定你能不能在真实项目和团队中持续做成事。

重点关注沟通、协作和推进能力：

- 沟通表达
- 文档写作
- 需求澄清
- 任务拆解
- 时间管理
- 复盘总结
- 协作意识
- 主动性
- 责任感
- 向上管理
- 跨团队沟通

## 总结

技术栈知识点决定你能不能做。

设计能力决定你能不能做对。

架构思维决定你能不能做大。

软技能决定你能不能持续做成。
