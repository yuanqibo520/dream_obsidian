关于人工智能在未来设计中的美学碰撞

最近在参与一个关于生成式AI的跨学科研讨会，产生了很多感触。当算法能够在一秒钟内生成出极其精美的视觉元素时，作为设计专业的学生，我们该如何重新定位自己的核心竞争力？

我认为，未来的设计不再仅仅是“创造形体”，更是“定义规则”和“赋予情感”。AI是一个极其强大的执行者，但它缺乏人类特有的直觉和对特定文化语境的深刻共情。
